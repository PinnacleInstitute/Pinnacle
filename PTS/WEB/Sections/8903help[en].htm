Custom Email Lists

The following custom email lists have been created for Pinnacle.  Email lists can be maintained with the Email Manager - "Mailing Lists" link in the upper right hand corner or the page.  The mailing list type must be set to ?Custom? and the desired "Custom List #" and optional parameters must be provided as specified below. Each email list will return up to five additional data items identified as data1, data2, data3, data4 and data5 for use in the subject and body of the email message.

Trial Memberships

Custom List #: 1
Parameter #1: Company ID (optional)
Parameter #2: Days Until Expiration (default=1 day before expiration) (negative value = days after expiration) 
Parameter #3: n/a
Parameter #4: n/a
Parameter #5: n/a

Email: Member Email
First Name:  Member First Name
Last Name: Member Last Name
Data #1: Company #
Data #2: Company Name
Data #3: Trial Expire Date
Data #4: Billing #
Data #5: n/a

How to use the ESN Search Function

{firstname},  This email is intended to help you more effectively search for documents and information on the ESN Search Engine.  

The search capabilities of the Search Engine has some advanced search functionality similar to Google. Using these techniques will help in searching for information. I have included a link to a page that includes the advanced search techniques. This link is also available from a help link next to the Search Box.  

http://www.pinnaclep.com/pagex.asp?Page=SearchHelp

Some of these advanced search techniques include:

Authorized Repair Centers - This separates each word as it's own search criteria
(ex. Authorized or Repair or Centers)
___

"Authorized Repair Centers"  - Returns Exact Phrase 
___ 

Reminder and messages - Returns everything where words appear next each other. 
___  

 "New *" - Returns every phrase that begins with the word
between in the quotations and in front of the (must have space) Asterisk  
___  

new near suitcase - Returns phrases where new appears near suitcase

___

Trade and not form - Returns phrases that contain Trade and not Form
(this is advanced and can be used as more of an advanced filtering)

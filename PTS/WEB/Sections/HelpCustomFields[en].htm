CUSTOM FIELDS

You can define custom fields to request on the Lead Program. Each custom field starts on a separate line and can continue to the next line if too long. The field name must be followed by a colon(:). If the name of the field starts with an Asterick(*), entry is required.

A list of choices must be separated by semi-colons(;).

If the name of the choice starts with an Asterick(*) it will be the default choice.

Non-list options can specify the data type (type=text|memo|number|date), minimum and/or maximum values (min=xxx max=xxx), width of text box or height of memo (size=xx), and a default value (value=abc). If a default value is specified it cannot include any spaces.

If type=date, the min, max and value settings can be entered as NOW for the current date.

A prompt can be included with the option name for input instructions (Age[Enter your Age]:) and cannot include a colon.

*Age: Under 20; 21 - 35; 36 - 50; Over 50  

*Income: *Under $40K; Under $70K; Over $70K  

Initials[Enter up to 3 characters]: type=text max=3 size=3  


Embedded Fields in the Page Content

You can optionally design your page content with the inputs for the custom fields built in the content rather than having them displayed after the content.  The following tips will help you setup embedded inputs.

  1. Remove all <FORM> tag definitions (and closing tags)
  2. Edit the name of each <INPUT> to be the exact name in your custom field definition.
  3. Edit the Submit button to include onclick="doSubmit(1,' ')"
  4. Edit <SELECT> <OPTION> tags to have a value = the text choice

EXAMPLE

Custom Field Definition

FirstName:type=text
LastName:type=text
Email:type=text
Phone:type=text
ColorList:Red;Blue;Green

HTML Source Code

<TR>
   <TD>First Name:</TD>
   <TD><INPUT type="text" name="FirstName"/></TD>
   <TD>Last Name:</TD>
   <TD><INPUT type="text" name="LastName"/></TD>
   <TD>Email:</TD>
   <TD><INPUT type="text" name="Email"/></TD>
   <TD>Phone:</TD>
   <TD><INPUT type="text" name="Phone"/></TD>
   <TD>Pick a Color:</TD>
   <TD>  
      <SELECT name="ColorList">
         <OPTION value="Red">Red</OPTION>
         <OPTION value="Blue">Blue</OPTION>
         <OPTION value="Green">Green</OPTION>
      </SELECT>
   </TD>
</TR>

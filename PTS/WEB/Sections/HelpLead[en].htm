Share your Lead URL with Everyone
You can promoted this lead program by using a variety of methods including, handouts, mailers, networking meetings, business cards, discussion boards, online communities, email, etc.
 
Personalized to You
The URL above for this lead program includes a parameter that uniquely identifies you. This lead program will be personalized to you. All new leads generated from this program will be assigned to you and added as a new prospect in the Sales Management System.
 
Your Very Own Domain Name
You may also want to purchase an inexpensive dedicated domain name for easier access that is redirected to the URL above. Contact customer support for assistance with purchasing and setting up a dedicated domain name.

URL Parameters

p = Lead Program #
m = Member #
a = Affiliate #
r = Prospect #
l = Language (optional - default = en)
x = Extra Data (optional)

Extra data can be anything you want to track. (e.g. where an ad was placed, who referred the lead, etc.).  The extra data is stored in the new prospect description.  Don't forget to use an ampersand (&) to separate parameters.

Tracking Your Advertising Campaigns
You can use the Extra Data parameter to keep track of which link a new prospect clicked to view your lead program.  Append the x parameter to your custom URL listed above to identify which advertisement the user responded to. 

For example, if you placed an ad in an online newsletter named PetCare.com, you may want to add an extra parameter to your custom url after your member number (e.g. 000) to identify PetCare as the source of the advertisement.   

http://www.pinnaclep.com/lp.asp?p=1&m=000&x=PetCare

The new prospect description tab will identify "PetCare" as the extra data.

Custom Link Message
You can use Microsoft Word to include your link in your own text message. Type your message in word and select all text to include in the link. Click the right mouse button and select the Hyperlink option.

Enter the desired link above and add any extra data using the "x" parameter as described above in the "URL Parameters". 

For example; http://www.pinnaclep.com/lp.asp?p=1&m=1&x=MyExtraData

After creating the Hyperlink in Word you can cut and paste it where ever needed like in your email signature.

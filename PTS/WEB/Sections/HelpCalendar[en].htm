BETA! Calendar


Overview
Each member and the company can have multiple Calendars. The Calendars can contain created appointments and optional items including committed goals, sales events, sales activity notes, completed classes, completed assessments, project due dates, and task due dates. The Calendar can be kept private or shared with the public and can be accessed anytime by the member’s mentor. 

Calendar Navigation
The current month and year is displayed near the top of the Calendar. The arrows on either side of the month and year can be clicked to view the previous year or month and the next month and year.

Information Navigation
Each of the displayed items on the Calendar can be clicked to edit the selected item. The appropriate page will be displayed based on the type of item. Clicking on an appointment will popup the appointment page and clicking on a goal will popup the goal page for editing.

Mentoring
The Calendar is an excellent tool for a mentor to view all the accomplishments, goals and activities for those that they mentor. Select the “Options” button near the top of the page to select the items to be displayed on the Calendar. All changes that are made to the Calendar options will be stores permanently with the Calendar.

Appointments
New appointments can be created by clicking the “New Appointment” button near the top of the page. Enter the new appointment name, location, description and start and end date and time. The appointment can be specified as an all day appointment. Optionally select the type of appointment and the importance and display (“Show”) options to the public. Each appointment type has a corresponding icon displayed on the Calendar. The importance and show settings also display an icon for non-normal selections.  If the Appointment has a location or notes, an ellipse is displayed after the appointment name.

Appointment Reminders
You can request the system to notify you via email before the appointment time. Select the desired amount of time before the appointment to be reminded (from 30 minutes to 2 weeks) and specify an email address for the reminder. To specify a custom reminder date, select no reminder and enter the custom reminder date and time in the “Reminder Date” field.

Recurring Appointments (Coming Soon)
You can specify that an appointment recur at a specific time for a specified period of time.

Calendar Refresh
As new appointments are created, existing appointment are modified, or any of the other displayed items are updated, the displayed Calendar can be refreshed by clicking the “Refresh” button near the top or bottom of the page.

Calendar Display Options
The Calendar name, time zone, public access and description can be edited by clicking the “Options” button near the top of the page. You can also select the type of items to display on the Calendar. Select the desired check boxes to display committed goals, sales events, sales activity notes, completed classes, completed assessments, project due dates, and task due dates. Each display item has a corresponding one-letter icon displayed with it on the Calendar for easier recognition. Icons for the selected items are displayed at the bottom of the Calendar as a legend.

Edit Appointments
The Calendar appointments and other items can be edited by clicking the name of the item. An appropriate page will be displayed for the selected appointment or other item. As appointments are updated the page will display an “Updated” notice in red at the top of the page. The appointment page will not close until you click the “close” button on the page. Click the Calendar’s “Refresh” button to display the updated appointment changes.

Daily View
You can view a more detailed listing of the appointments for a specific day by clicking the day number on the Calendar.

Multi-day Appointments
An appointment that spans multiple days is displayed in purple on all days that it appears within the current calendar.

Public Calendars
A Calendar that is specified for access to the public on the Calendar options page has a web address displayed at the bottom of the Calendar. A Calendar accessed by the public will only display appointments and no other items. Clicking the name of the appointment will display a popup window with additional detailed information. Public Calendars cannot be edited nor do they display the daily view.

Multiple Calendars
A member of the company can create multiple Calendars by clicking the “New Calendar” button near the top of the page. When multiple calendars exist they will be displayed in a list near the top-right of the page. Select one of the listed Calendars to display it with all off its selected display options for the current month.  The Calendars are listed in alphabetical order and the first Calendar in the list is initially displayed.  Update the name of the Calendars to list the preferred initial Calendar first, for example “1. My Calendar”.

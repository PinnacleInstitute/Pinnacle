Sales Management System

User Guide

Overview
The Sales Management System (SMS) is a sales tool designed specifically for tracking the process of converting a lead to a customer.  Many other tools are also accessible from SMS including Lead Programs, Affiliate Partners, and your Marketing Strategy.

Sales Tracking
The Sales Management System (SMS) provides members with powerful lead tracking capabilities and management reports. Each signed-in member that has the Sales Management System enabled can access it by clicking the “My Sales” link on the navigation bar on the left hand side of the page.

The SMS page provides links to all SMS functionality and the ability to search for prospects by prospect name, contact name, prospect #, email address and suspect date.  The contact name is searched by last name and first name (i.e. Smith, Bob).  A partial search word will return all matches that start with the search entry, except for prospect name, which returns all matches that contain the search entry anywhere in the name.  The search list results are sorted by the search field selected in the first combo box.  The prospect search can also be filtered by one of the predefined statuses by selecting a status option from the second combo box.  This is convenient for members to easily view their current suspects.

Sales Campaign
The sales campaign for the prospect is displayed at the top of the prospect page.  Click the “Change” button to select a different sales campaign for the prospect.  The sales campaign defines the sales steps that appear in the prospect status list, in the milestone date list, and on the Sales Board.
 
Prospect Type
The list of prospect types is a custom list defined by each company.  Select the appropriate prospect type or contact the company administrator for additional list options. The prospect type is used for information purposes and for categorizing prospects for management reporting.
 
Prospect Potential Value
The potential value of the prospect is used by the Sales Board to calculate the current value of all active prospects.  The conversion rate for the current sales step (status) will be multiplied by the prospect's potential value to calculate an estimated current value on the Sales Board.  The potential value may be the total value over a specific period of time (e.g. 1 yr., 2 yrs., 3 yrs.) for subscription services.
 
Prospect Status
Each prospect has five predefined statuses including, Suspect, “First Appointment”, Fallback, Closed and Dead.  The custom sales steps for the sales campaign assigned to the prospect will also appear in the status list after the predefined “First Appointment” status.

RSVP Response
The RSVP field is used to store the prospect’s response to an email message.  The RSVP response is also automatically logged in the notes by the system.  The logged RSVP response is defined in the email message for an automated email sales step.  The RSVP response may be used for management reporting statistics on automated email campaigns.

Agreed Upon Next Step
The sales person should track the next step that the prospect has agreed upon.  There should always be an agreed upon next step with a specific date and time.  Any prospect without an agreed upon next step with a specific date and time should be considered in a Fallback status.  The next events are selected from a predefined list that includes; Call, Meeting, Email, Mail, Drop By, and Other.  Click the “Clear” button to reset the Next Event, Next Date and Next Time fields.
 
Milestone Dates
All five of the predefined sales steps include a milestone date to track when the event happened.  Additionally each custom sales step that has a milestone # indicated will also appear in the list of milestone dates.  A short description follows each milestone date field.  Click the calendar icon following each milestone date field to select a date from a popup calendar.
 
Auto Change Status
Automated sales steps set the “Auto Change Status” and change date fields.  These fields may also be manually set by the salesperson.  The system checks every hour for prospects that have a status change date specified.  If the new sales step being changed to specifies a next step, the system will set the prospect’s next status and date.  The change date can contain a date and time (e.g. 1/1/05 1:00pm)

Auto Email Message
Automated sales steps set the Email message and send date fields.  The email message # may be edited by management.  If an auto email message is specified the user can edit the send date.

Contact Information
In addition to the prospect name, SMS also tracks the contact name, title, email address, 2 phone numbers, and street, city, state postal code and country.  The prospects web page and a 2000 character description may also be entered.  If the prospect is a business, the business name is entered as the prospect name and the primary contact is entered as the contact name.  If the prospect is an individual, their name is entered as the contact name and the prospect name can be their last name and first name for searching.  Additional contact information may be entered in the description field or in the notes.
 
Prospect Notes
Multiple notes may be stored for each prospect. Each 1000 character note tracks who made the note and when the note was made.  Automated sales steps and email RSVPs also add notes to document the activity.  Notes can be viewed and added by selecting the “Notes” link at the upper right hand corner of the prospect page.  A note may be pinned to the top of the list for priority access to important information.
Prospect Documents
Multiple documents may be attached to each prospect. Documents can be viewed and added by selecting the “Documents” tab near the right side of the page. 
 
Prospect Shortcuts
Members and Administrators can create shortcuts to prospects by clicking the shortcut icon   that follows the prospect name at the top of the page.  The user can view a list of all their shortcuts by selecting the shortcut link at the top of the navigation bar on the left hand side of the page.  Shortcuts may be pinned to the top of the list for priority access.  Pinned shortcuts will also appear as large icons on the member’s home page.

Prospect Lookup
Members can lookup a potential suspect before entering it into the system to check if another member is already working with the prospect.  If the suspect exists, the member can view the basic contact information for the member to possible contact him/her with additional assistance or partnership.  Select the “Prospect Lookup” link at the top of the “Sales Management System” page.
 
View Sales Documents
If the current sales step of a prospect includes attached documents, a link is displayed at the top of the prospect page.  Selecting the link will display a popup window with a list of all attached documents.  Clicking the name of a document will display the document in a separate popup window.  Sales Documents are convenient ways for salespeople to easily access sales scripts and reference documents. 


Sales Management

Management Reports
Sales people and management can review the following sales reports.  Sales people only view prospects assigned to them.

1. SALES BOARD:  Sales Board of prospects at each sales step with estimated sales value.
2. SCHEDULE:  Schedule of events to be performed for the specified time frame.
3. FIRST APPOINTMENTS:  Statistics on first appointments set for the specified time frame.
4. FALLBACK:  List of prospect in fallback status for the specified time frame.
5. CLOSED:  List of closed prospects for the specified time frame.
6. CONVERSION RATES:  Member conversion rates for each step in a campaign. (coming soon)
7. EXCEPTIONS:  List of prospects that have an exception like a past next event or a current status that is past the recommended length of time. (coming soon)

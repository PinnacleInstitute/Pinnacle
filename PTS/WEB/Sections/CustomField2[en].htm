Custom Fields

You can define multiple custom fields to manage extra information. Each user-defined custom field starts on a separate line and can continue to the next line if too long. The field name must be followed by a colon(:). The following options are available.

> > > > > > > > > > > > > > > > > > > > > > > > > > > > > > > >
OptionDescription
Options preceeding field Colon:
*Required Field
-Private Field
_Read Only Field
?Search Field
+Combine with previous Field
{x}x = Field id
[x]x = Field prompt (following field name)
Options following field Colon:
;identifies list items (leading * specifies default choice)
type=xx = text (default) | memo | number | date | yesno
min=xx = minimum input value
max=xx = maximum input value
size=xx = width of text box or rows for memo field
value=xx = default value (spaces not allowed) (NOW option for date fields)

EXAMPLES

*Size: Small; Medium; Large; Extra Large  

*Color: *Red; Green; Blue  

Initials[Enter up to 3 characters]: type=text max=3 size=3

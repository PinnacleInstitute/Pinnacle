Domain Forwarding Tutorial

 

Objective: Understand how to forward domains on GoDaddy.com

Now you are ready to set up a Domain forwarding address for your newly created Domain name.

If not already logged on, Enter Customer# or Login name and password in top right corner, and click 'Secure Login'. 

If already logged on, then select the "My Account" option under the "My Account" menu.  [See example at right]

In the Account Manager, under the bold text Domain Names, choose Manage Domains.  [See the Manage Domains menu by clicking here]

Basic instructions for domain forwarding on GoDaddy.com are as follows:
 1. Select the domain name(s) that you want to forward and click Forward Domains.

 2. Indicate if changes apply to only the Single Domain, Selected Domains, or All Eligible Domains.
• Single Domain is the domain that you have selected in the Domain Name list.
• Selected Domains appears only if you have more than one domain selected.
• All Eligible Domains are all active domains that are not pending, private, or locked.

 3. Enter the URL of the Web site that you want the domain(s) to be forwarded to. 
     For example:   http://www.pinnaclep.com/lp.asp?p=5&m=0
  URL Parameters:
p = Lead Program #
m = Member #
a = Affiliate #
l = Language (optional - default = en)
x = Extra Data (optional)

 4. Indicate if you want to apply masking to the domain name. (see user guide link below for more info on masking)

 5. Click Save Changes . [Make note of the message that appears: "Your changes have been submitted... Domain forwarding changes can take up to 24 hours to take effect."]

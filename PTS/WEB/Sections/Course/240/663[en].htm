Changing Document Views

Word provides several different ways in which you can view your document. Document views can be changed using the View buttons located at the left end of the horizontal scroll bar. You should select the view that allows you to work best with your document.

Normal view is the best view for typing, editing, and formatting. This view is the one that many Word users prefer.

Web Layout view is used for creating and editing web pages. Documents in Web Layout view behave like browsers where the text wraps to fit the window.

Print Layout view provides a WYSIWYG (What You See Is What You Get) view of a document. Some commands, such as inserting text or headers and footers, require you to work in this view. This view is the default view for a new installation of Word.

Outline view is useful in long documents that use a heading style. Outline view allows you to expand and collapse outline levels so that only certain heading levels appear. This view is helpful in a document containing many heading levels. The Outlining toolbar automatically appears in this view. This toolbar allows you to easily reorganize a document by moving information and adjusting indent levels.

You can also switch views by selecting the desired view command from the View menu.When you save a document, the selected view is also saved. The next time you open the document, it will open in the same view.

Changing Document Magnification (Zoom)

You can magnify a document to make it easier to read, or you can reduce the magnification in order to see more of the document.

By selecting the Zoom command on the Edit menu, you can set the magnification level to a percentage of the actual size, or you can choose to view the entire width of the page. Changing the magnification does not change the document; it only changes how the document appears on the screen.

You can also change the magnification of a document using the Zoom list on the Standard toolbar.

    Displaying/Hiding Rulers

    Many times, it is helpful to use the ruler for referencing locations. You can display the horizontal ruler in Normal and Print Layout view. You can use the horizontal ruler to set left and right margins and indents, as well as to add and remove tab stops.

    You can hide the ruler if you want to be able to view more of your document. The Ruler command on the View menu is a toggle, which means that the same command enables and disables the feature.

    In Print Layout view, you can also display a vertical ruler. You can use the vertical ruler to set top and bottom margins and to place items at specific locations in the document. Hiding the ruler in Print Layout view also hides the vertical ruler. Select the Options command on the Tools menu and then enable the vertical ruler under Print and Web Layout options on the View tab of the Options dialog box.

Opening Multiple Documents and Switching Among Them

You can open multiple documents in Word. When you open more than one document in Word, the first document appears within the application window with its own document Close Window button. All other open documents, however, appear in a Single Document Interface (SDI), meaning that each open document has its own menu, toolbar, and application window. In addition, each document gets its own button on the taskbar. In other words, each document appears within its own Word application.

You can open multiple documents in the Open dialog box with the [Shift] key or the [Ctrl] key. When you use the [Shift] key, all files between the first and last selected files in the dialog box are selected. When you use the [Ctrl] key, you can select non-adjacent files in the dialog box.

When you open multiple documents in Word, the names of all open documents appear on the Window menu with a checkmark indicating the active document. They also appear on the taskbar with the active document taskbar button selected.

There are several reasons for opening multiple documents, including copying and pasting text between them as well as comparing their contents.

If you have more than one open document, you can use the Window menu or the taskbar to quickly switch between them.

Learn to sling the "Web terms" like a pro.

This new and exciting technology has also brought with it a new terminology. Whether you aspire to a career as a Web developer or plan on using the Internet as an online storefront, you need to be able to sling the "lingo" with other Web professionals.

Lets begin with all the "Web Words"

Web Page
A Web page is a combination of words and images that has been formatted for display on a computer monitor.

Web site
A collection of Web pages stored in a specific location on the World Wide Web. Each Web site contains a home page (usually named "index.html", which is the first document users see when they enter the site. The site might also contain additional documents and files including graphics, audio, video, cgi scripts, and databases.

Webmaster
An individual who manages a Web site. Depending on the size of the site, the Webmaster might be responsible for any of the following:

  • Making sure that the Web server hardware and software are running properly
  • Designing the Web site
  • Creating and updating Web pages
  • Replying to user feedback
  • Creating CGI scripts
  • Monitoring traffic through the site

The exponential growth of the World Wide Web has created an unprecedented demand for Webmasters. There may be several aspiring Webmasters enrolled in this course.

World Wide Web
Also known as the WWW, the W3, or most often simply as the Web, it originally developed by CERN labs in Geneva, Switzerland. Continuing development of the Web is overseen by the World Wide Web Consortium.

Internet
The Internet originated in 1969, in the midst of the Cold War, as a "nuke-proof" communications network. It consists of countless networks and computers across the world that allow millions of people to share information. The lines that carry the majority of the information are know as the Internet backbone. While the government used to run things, now major Internet service providers (ISPs) such as MCI, GTE, Sprint, UUNET, and ANS own portions of the backbone- a good thing as they have the motivation and the revenue to maintain the quality of these large networks.

URL
uniform resource locator
URLs are the Internet equivalent of mailing addresses. How do they work? Like other types of addresses, they move from the general to the specific (from zip code to recipient, so to speak). Take this URL, for example:
http://www.sonic.net/~webclass/video

First you have the protocol:
http:/

then the server address or domain:
/www.sonic.net

and finally the directory:
~webclass/video/

Note - there is no specific file included in this URL. If a file called index.html resides in the video directory it will automatically be opened when the URL is accessed..

Is URL pronounced "you are ell," or does it rhyme with hurl? Both pronunciations are widely used.

Domain Name
Looking for a domain name? You'll find it to the right of the @ sign in an email address, or about ten characters into a URL. Our Web site's domain name is
www.sonic.net.

Domain names are issued by the National Science Foundation (NSF). They assign different extensions based on whether the domain belongs to a
commercial enterprise (.com)
educational establishment (.edu)
government body (.gov),
the military (.mil)
a network (.net)
nonprofit organization (.org).

Headings work great for identifying major sections of your Web page, however sometimes you need to change font size without adding a line break or bolding text. The <FONT> tag can control the size, color, and shape of your text.

Let's get started.

Step 1 - Change Text Size using <FONT> tag and the "size" attribute
Let's change the font size of our corporate headquartes address. Font sizes range from 1-7 with 1 being the smallest and 7 being the largest. If this sounds like the opposite of how headings behave, it is...

Add this code
<FONT SIZE="2">Corporate Headquarters:</FONT>

The default size for text is 3. By changing the size to 2 it will become smaller. When I change font sizes the text does not break or become bold, I can have several font sizes on the same line. I have decided to display all of the contact information in font size 2 so my code will look like this.

<FONT SIZE="2">
Corporate Headquarters:
<br />
Burnt Pan Productions
<br />
01010 Digital Drive
<br />
Etherware, CA 10101
<br />
(800) 123-4567
</FONT>

Notice I turned on the Font tag before typing "Corporate Headquarters" and then turned it off several lines later after the phone #. You do NOT need to declare a font size for every line of text. You also do not need to include the attribute size=2 in the ending font tag. When you end the font tag all the attrbutes are ended as well.

Step 2 - Controlling the Shape of Text using the <FONT> tag and the "face" attribute
If you have a desktop publishing background, this step sounds pretty exciting. Well... there are a few things you should know. You can identify any font face you want but it will display only IF the computer displaying your page has that font installed. So, if you declare this tag

<FONT FACE="arial, helvetica, sans-serif">

Chances are good that someone viewing your Web page in Kansas will display the text with the correct font. However if you are counting on that same computer having "Comic Sans" or "Playbill", you may be disappointed. If you declare a font face which is not installed on the users computer, the browser usually substitutes "Times Roman".

Lets change the font face for the entire page. After the <BODY> tag insert this code

<FONT FACE="Arial, Helvetica, sans-serif">

This turns on the font face "arial". At the end of the document I will end the font tag

</FONT>
</BODY>
</HTML>

Step 3 - Changing Text Color using the><FONT> tag and the "color" attribute
For the copy of your Web page, I recommend Black text on a White background. However, there may be situations where you want to emphasize text by changing the color. Let's say I want to change part of the company slogan to a rust color. I add the color attribute to the font tag.

move over Pet Rock...&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <FONT color="#993300">there's a new Pet in town</FONT>

So where did 993300 come from? Well... that is a Hexidecimal number which controls the color displayed by the browser.

Many people surf the net with a monitor set to read only 256 colors. If you specify a color that is not one of the defined browser safe colors, then the browser will display either the closest color, or a dithered (spotty) color. There is a palette of the 216 Browser Safe Colors which can be displayed by all Windows and Macintosh computers. These colors can be declared using Hexadecimal values. To find out what those values are use the Browser Safe Color Palette found at the visibone site.

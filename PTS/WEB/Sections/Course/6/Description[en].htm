Course Description
We will discuss WWW terms and how HTML works. Students will be asked to evaluate Web sites created by students who have completed the Advanced HTML courses and incorporate ideas from these sites into their own.

Students will use HTML to create a web page that has the following elements: basic tags, text in various size and colors, text forrmatted in bold and italic.

You may be wondering why you should even bother to learn HTML, with all these slick new Web editors on the market, why bother? In order to effectively use Dreamweaver or FrontPage you need to understand the fundamental principles of HTML.

A Web editor is like a calculator and HTML is like math fundamentals. You wouldn't use a calculator without a background in mathematics, don't begin to build a Web site without a background in HTML.

 
Course Requirements
A personal computer, an Internet connection, a Web browser (Internet Explorer or Netscape) and a text editor such as Notepad or SimpleText.
 
Course Goals
Here is what you will acheive by the end of the course:

* understanding of some basic Web Terms

* What is HTML and how does it work?

* a web page which contains "basic HTML tags"

* Text in three or four different sizes, colors and styles

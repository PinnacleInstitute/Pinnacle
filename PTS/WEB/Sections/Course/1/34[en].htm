You can predefine email templates to be used when sending lead pages, sales presentations or personal emails to contacts, prospects and customers.  


 

Custom Email Templates

 

Custom Email Templates are predefined for use when sending a lead page or a sales presentation to a prospect, or when sending a personal email to an individual contact, prospect or customer.  Click the "Add New Email Template" option in the upper-right corner to define a new email template.  Click the pencil icon following the template name in the list to edit an existing email template.

 

 

 

 

Edit Email Template

Click the first pencil icon following the name in the list to edit the existing email template.  You can create a custom email message with the WYSIWYG (What-You-See-Is-What-You-Get) Editor to apply formatting similar to a word processor.  You can use the three editor toolbars to select the font style, color and size, format and align the text as well as many other functions.

REPLACEABLE TAGS

The following tags can be placed in the email message to be replaced with the appropriate values when used.

 

{p-firstname} prospect's first name
{p-lastname} prospect's last name
{p-email} prospect's email address
{p-phone} prospect's phone #1 (Day Phone)
{p-event} prospect's next event
{p-id} prospect's unique identifier
{m-firstname} member's first name
{m-lastname} member's last name
{m-email} member's email address
{m-phone} member's phone #1 (Day Phone)
{m-id} member's unique identifier
{signature} member's signature

 

The email template name and category can be edited at the bottom of the page.  The status must be active to appear in the list of available email templates.

 

 

 

Email Signature

It is recommended to use the {signature} tag as your email signature.  This tag will be replaced with the appropriate signature from your custom defined signatures.  Review the "Custom Signatures" tutorial to learn more.

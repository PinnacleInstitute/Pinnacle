You can create web pages that can contain any information that you would like to use or share with others.  


 

Custom Information Pages

 

You can create an unlimited number of custom Information Pages containing any web page content.  Click the "Add New Information Page" option in the upper-right corner to create a new information page.  Click the pencil icon following the page name in the list to edit an existing information page.

 

 

 

  

Edit Page Content

You can edit the content of each of the information web pages in the list.

There are several icons to assist you with editing the page content.  The preview  icon on the right side of the list will display the selected page.  The upload  icon will allow you to upload a web page that you created on your personal computer with your HTML webpage editor.  This option supports advanced editing of content that cannot be edited in the included simple WYSIWYG editor.

You can use any HTML editor like Microsoft FrontPage, Microsoft Expression or Adobe Dream Weaver.  If you do not have an HTML webpage editor you can download a free and professional WYSIWYG HTML editor at  http://www.kompozer.net/download.php.

After you've completed your editing of the web page you can upload the page by clicking the upload  icon.  The uploaded file must have a *.htm extension.

All resources in your web page must be accessed on the server including custom images, CSS style sheets, Flash movies, etc.

 

 

Edit Information Page

Click the first pencil icon following the page name in the list to edit the existing information page.  You can create a custom web page with the WYSIWYG (What-You-See-Is-What-You-Get) Editor to apply formatting similar to a word processor.  You can use the three editor toolbars to select the font style, color and size, format and align the text as well as many other functions.

The information page name and category can be edited at the bottom of the page.

Each information page has its own web address (URL) displayed near the top-right corner.  You can include this web address in other information pages to link the pages together.  

Other web resource that you may need to include in your information page (e.g. images, flash, etc.) can be uploaded in the custom documents section of the Resource Center.  Each uploaded document has it's own web address for referencing in your information page.

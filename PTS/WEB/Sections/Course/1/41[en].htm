You can create your own Custom Presentations to qualify serious prospects.


 

Presentations

A list of all your Custom Presentations will be displayed for your review and editing.  The Presentation status is displayed on the right side of the list.  You can click the Preview, Traffic and Usage icons in the middle of the list to review the corresponding functionality of your Custom Presentations.  You can click the pencil  icon to the right of the Custom Presentation name to edit the page options.

Click the "Add New Presentation" link in the upper-right corner to add a new Presentation.



 

Add Custom Presentation

Enter a name for your new Presentation.  You can easily create a new Presentation by copying an existing Presentation from the list box at the bottom of the page.  Click the "Add" button at the bottom of the page to add your new Presentation.

You can view a list of custom presentations from any other group by entering the desired group number in the text box below the list and clicking the "Reload" button.

Your newly created Presentation will contain a copy of all the content from the copied Presentation.

Edit Custom Presentation

In your list of Custom Presentations, click the pencil  icon to the right of the Custom Presentation name to edit the page options. You can edit the Presentation name and set the status to active when you have completed editing the Presentation and it is ready to be used.  

You can edit the page options and content at the bottom of the page. 


Your Presentation actually consists of multiple web pages.  The first ones are  the presentation pages that will be displayed to the visitor.  They contain the complete presentation of the value proposition that you are promoting to the target market.  Now that you have their interest, they will spend the time to review your presentation.  

The last page is the default email template that is used when you email this Presentation to someone.

 Click the pencil  icon to the right of the page name to edit the page options.

  

Edit Page Options

You can edit your page name, number, status and language for the current page.  

  

Edit Page Content

You can edit the content of each of the web pages that makeup your Presentation from the page list at the bottom of the edit page.

On the right side of the list are several icons to assist you with editing the page content.  The preview  icon will display the selected page.  The download  icon will allow you to save the page to your personal computer for editing with your HTML webpage editor. 

You can use any HTML editor like Microsoft FrontPage, Microsoft Expression or Adobe Dream Weaver.  If you do not have an HTML webpage editor you can download a free and professional WYSIWYG HTML editor at  http://www.kompozer.net/download.php.

After you've completed your editing of the web page you can upload the page by clicking the upload  icon.  The uploaded file must have a *.htm extension. 

To assist you with the editing of your web pages you can click the Stock Images icon above the page list.  This will display a list of images that are available for your use on your web pages.  Copy the URL included with each image to your HTML editor image source.  You may or may not have the Stock Images option.  

You may also upload your own custom images by clicking the Upload Files icon above the page list.  This will allow you to upload your own images to the server for their use in your web pages.  Copy the URL specified with the uploaded file to your HTML editor image source.

All resources in your web page must be accessed on the server including stock images, custom images, CSS style sheets, Flash movies, etc.



 


  

Edit Email Page

The email page includes an extra edit  icon in the page list. Click this "Edit Page Content" icon to edit your page content with a simple WYSIWYG (What-You-See-Is-What-You-Get) Editor to apply formatting similar to a word processor.  You can use the three editor toolbars to select the font style, color and size, format and align the text as well as many other functions.  

 

  

Replaceable Tags

You can insert tags that will be replaced with your's or the prospect's personal information.  The following replaceable tags are supported by the different page types.

The most useful replaceable tag is the {signature} tag.  It will insert the appropriate Presentation signature from your list of defined signatures.

Presentation Page
{signature} your Presentation signature
{name} your first and last name OR company name if company
{email} your email address
{phone} your day-time phone number
{image} your uploaded image 
{id} your internal id number
{ref} your internal reference number
{p-firstname} prospect's first name
p-lastname} prospect's last name
{p-email} prospect's email address
{p-phone} prospect's day-time phone number
{p-id} prospect's internal id number


Email Page
{signature} your email signature
{m-firstname} your first name
{m-lastname} your last name
{m-email} your email address
{m-phone} your day-time phone number
{m-id} your internal id number
{p-firstname} prospect's first name
p-lastname} prospect's last name
{p-email} prospect's email address
{p-phone} prospect's day-time phone number
{p-id} prospect's internal id number
{link} system generated link to Presentation

The {link} tag in the email must not be modified. It will be replaced by the system with the system generated link to the Presentation with all the required parameters including which Presentation, your id and the prospect's id if known.  


Presentation Page Links

You can provides links between each of the pages in your presentation with the following required URL.

pp.asp?pg=[x]

where [x] is the page number (without the brackets) on the left side of your page list.  Using this link will enables the system to track which pages the prospect views.

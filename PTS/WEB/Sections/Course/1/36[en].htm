You can upload custom documents that can contain any information that you would like to use or share with others.  


 

Uploaded Documents

 

You can upload an unlimited number of custom documents containing any content.  Click the "Add New Document" button to create a new custom document.  Click the pencil icon following the page name in the list to edit an existing document.  

 

 

 

  

Edit Document

Click the pencil icon following the name in the list to edit the existing document.  The document may be an uploaded document from your personal computer or it may be a reference to a document anywhere on the Internet.

Each uploaded document has its own web address (URL) displayed in the middle of the page in purple.  Your uploaded documents can be referenced in your information pages or anywhere else a URL can be used.

Click the "Upload" button to select a file from your personal computer to be uploaded.  You may alternately specify a fully qualified URL starting with "http://" in the "Upload Link" text box.

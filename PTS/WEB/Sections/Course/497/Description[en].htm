Technical Requirements

IBM-compatible PC with 166 MHz or higher processor, 32 MB RAM (64 MB RAM recommended); 800x600 SVGA color monitor with 16-bit high color display; Internet Explorer 4 or Netscape Navigator 4 or later with Shockwave 7.0.1; Windows 95, Windows 98, Windows NT; 3.5 diskette (for tracking); 28.8 kbps connection or higher for on-line use.

Course Requirements

Individuals taking this course should have a working knowledge of Lotus Notes (prior to Release 5.) In addition, they should also have knowledge of Microsoft Windows 95 (or higher) or NT Workstation and should have experience using a mouse in a graphical user interface environment.

Course Goals

Upon completion of this course, the student will be able to:

Navigate easily through the new Notes format using the new task buttons, shortcut keys, and navigation buttons
Create bookmarks and bookmark folders
Customize your Welcome page
Locate addresses in the new Name and Address Book and from Domino directories
Create new memos and custom stationery
Set mail preferences, and filter incoming mail
Schedule meetings, set up your free time schedule, and work with group calendars
Print calendars

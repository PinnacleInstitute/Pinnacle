Technical Requirements

Netscape or Internet Explorer (versions 4.0 or higher).  Browsers should be capable of accepting cookies; SVGA Hi-color mode 800x600 resolution display.  Macromedia Shockwave 6.02 or higher.

Course Requirements

Before taking this course the user should be familiar with the basic knowledge of HTML.

Course Goals

Analyze requirements for developing a Web site Identify the steps in a Web site development process Identify the features of FrontPage Install FrontPage 2000 Identify the views in FrontPage

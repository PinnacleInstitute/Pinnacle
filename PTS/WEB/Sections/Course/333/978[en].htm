The Lookup Wizard
Step 1: Create a Lookup Table

Some fields in tables require that data be from one body of data. For example, for customers living in the U.S., the State field will be limited to the 50 states. You can use an option called the Lookup Wizard to provide the possible choices for the user, so that instead of having to type the data, the user can select it from a list. To create a lookup table for a field, you must be in Design view. Click the drop-down arrow on the Type field and choose Lookup Wizard.
Step 2: Choose to Type Your Own List

To create your own list of options for the user to choose from, click the “I will type in the values that I want” option and then click the Next button.
Step 3: Type the List

The next dialog box of the Lookup Wizard provides space to type the list of options for the user. For example, you could type the two-letter abbreviations for the fifty states. After typing your list, placing each option on a separate line, click the Finish button.
Step 4: Try Out the Lookup Table

Click the View button to return to Datasheet view. Enter a new record into the table, and when you get to the field with the lookup table, click the drop-down arrow and choose one of the options. The data from the list is placed in the field.

Text & Memo Data Types
Step 1: Use Design View

If you are working with a table in Datasheet view and find that you want to include a long text field in the table, click the View button to switch to Design view.
Step 2: Name the Field

If you have included a text field in a table, the maximum size can only be 255 characters. If you would like to type a description of data, Access provides a second option, called a Memo field type. You can type sentences and paragraphs up to a maximum of 64,000 characters. To add a Memo field, click in the Field Name column at the bottom of the list and type the field name.
Step 3: Select Memo

Click in the Data Type column and then click the drop-down arrow to display the type options. Click the Memo option.
Step 4: Switch to Datasheet View

From Design view click the View button. The button appears with a drop-down arrow to display the text name for the button as well as the icon. You are prompted by Access to save the table design. If you choose Yes, the changes you made will be saved and displayed in Datasheet view. If you choose No, the changes you made will not be saved, and the design in Datasheet view will not be changed.

Step 1: Use Design View

If you are working with a table in Datasheet view, click the View button to switch to Design view.

Hyperlink Data Type
Step 1: Use Design View

If you are working with a table in Datasheet view, click the View button to switch to Design view.
Step 2: Select the Field

Click in the Data Type column for the field you want make into a Hyperlink field. Use this data type to link to documents on your computer, the intranet in your business, or the World Wide Web. Click the drop-down arrow and select the Hyperlink option.
Step 3: Switch to Datasheet View

From Design view click the View button. The button appears with a drop-down arrow to display the text name for the button as well as the icon. You are prompted by Access to save the table design. If you choose Yes, the changes you made will be saved and displayed in Datasheet view. If you choose No, the changes you made will not be saved, and the design in Datasheet view will not be changed.
Step 4: Add a Hyperlink

Select a record in the hyperlink field where the hyperlink is to be added. Click the Insert Hyperlink button on the toolbar. The Insert Hyperlink dialog box displays. To create a link to a file on your disk, click the File button to locate the file. To create a link to a file on the World Wide Web, click the Web Page button to locate the file or type the URL of the document in the Type the file or Web page name text box.

Click in the Text to display text box and type the text that should displayed in the field. Click the ScreenTip button to open the Set Hyperlink ScreenTip dialog box. Type the text to display when the user points the mouse to the hyperlink. When you return to the Datasheet view, the information you typed in the text box appears in the field. When you point to the hyperlink, the mouse pointer changes to a hand, and the ScreenTip displays.

Create a Switchboard Form
Step 1: Open Switchboard Manager

Open the Tools menu, select Add-Ins, and then Add-In Manager. You will see a dialog box informing you that no switchboard exists for this database. Click Yes to indicate you wish to create one.
Step 2: Open Edit Switchboard Dialog Box

The Switchboard Manager dialog box appears with Main Switchboard (Default) listed. Click Edit to customize the Main Page. (If you're editing an existing Switchboard, select it from the list box, and then click Edit).
Step 3: Add an Item

Click New on the Edit Switchboard Page to add your first item to the Main Switchboard. In the Text box, type Enter/View Name of Form, substituting the name of the form you want to use. From the Command drop-down list, select Open Form in Add Mode.
Step 4: Choose a Form

From the Form drop-down list, choose the form you want to open. After this command is executed from the completed Switchboard, the form you specified will open, ready for you to enter data. Click OK to add the command to the Main Switchboard.
Step 5: Add an Exit Button

Include an Exit button so that you can close the database from the Switchboard. Click New again, type Exit in the Text box, and select Exit Application from the Command drop-down list. Click OK to return to the Edit Switchboard dialog box.
Step 6: Exit Switchboard Manager

From the Edit Switchboard dialog box, you can continue adding elements to your Switchboard as needed; use the Move Up, Move Down buttons to change the order in which the items appear. To exit the Switchboard Manager now, click Close, and click Close again.

Format a Switchboard Form
Step 1: Use the Property Sheet

To format or change an item on the form, such as the Switchboard label, double-click the item to open its property sheet (be sure to double-click the border of a selected item). The All tab has numerous properties you can edit for the item. Click a property to display a drop-down list or button for formatting the item.
Step 2: Change the Text

You can click in the Caption box and type a new caption, such as My Address Book or My Household Inventory. To change the font, scroll down the list and click the Font Name property to display a drop-down list of fonts. Click the Close button to close the property sheet.
Step 3: Move the Label

The new label text is now in place; to reposition the label, select it to display the selection handles. Position your mouse pointer at the edge of the label box until it turns into a black hand. Hold down the left mouse button and drag the label to a new location on the Switchboard.
Step 4: Change the Graphic

To change the graphic, double-click the Image box on the left side of the Switchboard form to open its property sheet. Click the Picture property line to access the Build button. Click the Build button to open the Insert Picture window.
Step 5: Choose a Graphic

Locate the graphic file you want to use and click OK. Close the property sheet to return to the form, and the new graphic is inserted.

Enter Data with the Switchboard
Step 1: Use the Buttons

Depending on your database, some fields on your form may include a drop-down arrow to select a category. Click the arrow to display the drop-down list and make a selection. Or, if your database doesn’t have drop down arrows, just click one of the Switchboard buttons.
Step 2: Fill Out the Form

A form opens for you to begin filling out. Depending on the type of database you're creating, your fields will vary. To enter data into a field, click inside the field and start typing. If you make a mistake, use the Backspace key to redo your typing. Press Tab to move from field to field.
Step 3: Enter a New Record

To enter another record, click the New Record button on the Database toolbar. Continue filling out the form for each new record you add.
Step 4: Close the Form

To close the form, click the Close (X) button in the upper-right corner of the form and return to the Main Switchboard.

Advanced Record Sorting
Step 1: Open the Table

From the Database window, click the Tables tab and open the table you want to sort (double-click the table name or select it and click Open).
Step 2: Choose the Sort Field

Click the column header for the field you wish to sort, or place your cursor anywhere in the column you want to use for the sort.
Step 3: Use the Sort Buttons

To sort the table in ascending (A-Z) order, click the Sort Ascending button on the toolbar. To sort by descending order (Z-A), click the Sort Descending button.
Step 4: The Data is Sorted

The data is immediately sorted. Continue sorting the table fields as needed. When you exit the table after sorting, a prompt box asks you if you want to save the changes; click Yes if you do, or No if you don't want the sort to be permanent.

Filter Records
Step 1: Filter by Selection

Open the form you want to use, and then click the field that contains the criteria you want to filter for. If you want to search for all the records using the same zip code, for example, first locate a record containing that zip code and click in the zip code field. To begin the filter, click the Filter by Selection button on the toolbar.
Step 2: The Records Are Filtered

Any records that have matching information in the field selected have been retained, but all others have been hidden (filtered out). Use the form's arrow buttons to view the filtered records.
Step 3: Remove a Filter

To remove the filter, click the Remove Filter button on the toolbar. (When no filter has been applied, the Remove Filter button does double duty as the Apply Filter button).
Step 4: Filter by Form

Click the Filter by Form button to enter your own criteria for the filter. A blank form appears as a single record in a datasheet.

Replace Record Data
Step 1: Open the Table

Click the Tables object in the Objects bar of the Database window. The tables in the current database are listed. Either double click the desired table, or click the table to select it and then click the Open button. Both methods open a table.
Step 2: Use the Replace Option

You can use the Replace option to quickly edit contents of multiple records. For example, if a telephone area code changes, you don't want to have to edit hundreds or thousands of records to make the change. Simply use the Replace command to find the area codes and replace them with the desired one. To locate the data to replace, first place the insertion point in the field you want to search.
Step 3: Use the Replace Dialog Box

To open the Find and Replace dialog box, choose Edit and select Replace. Type the value you want Access to find in the Find What field. Type the replacement data in the Replace With field.
Step 4: Replace Each Individually

If you want to be cautious with the table data, you can replace record data, one record at a time by clicking the Find Next button in the dialog box. And if you want to replace the data that displays, click the Replace button. You would need to repeat this process for all records in the table.
Step 5: Replace All Data at Once

If you know you want to replace all instances of the data in the table, and you want to do this quickly without looking at each record, you can click the Replace All button in the Find and Replace dialog box. Access will change every match of the data to the new data behind the scenes. When you are finished using Replace, click the Cancel button to close the dialog box.

Data Searching Options
Step 1: Open the Table

Click the Tables object in the Objects bar of the Database window. The tables in the current database are listed. Either double click the desired table, or click the table to select it and then click the Open button. Both methods open a table.
Step 2: Open Find or Replace

To open the Find and Replace dialog box, click the Find button on the toolbar or choose Edit, Find or Edit, Replace. The dialog box opens as shown in this figure. After typing the data to find or find and replace, click the More button in the dialog box.
Step 3: Set Search Options

You can set options to help narrow the search. Use the Look in option to change the field or table to search. Use the Match option to have Access match the entire field, match any part of a field, or match only the beginning of a field. You can use the Match Case option to limit the search for words with the same capitalization or to expand the search to find all instances regardless of the capitalization. Use the Search Field as a Formatted option to find only matches to the formatting you specify.
Step 4: Find or Replace Data

After specifying Find and Replace options, find records or find and replace data as you always do. If you have problems with the search, check that you specified all options correctly.

The Simple Query Wizard
Step 1: Open the New Query Dialog Box

From the Database window, select the Queries tab and click New to open the New Query dialog box.
Step 2: Start the Simple Query Wizard

Choose Simple Query Wizard and click OK to open the first Simple Query Wizard dialog box.
Step 3: Choose a Table

Open the Tables/Queries drop-down list and select the first table from which to choose a field.
Step 4: Choose a Query Type

Select a field and click the > button to add it to your query. Select as many fields from as many different tables as you wish and click Next when you're ready to continue.
Step 5: Give the Query a Name

The next dialog box asks you if you want a detail (shows every record) or summary (use for totaling values) query. Detail is the default, so click Next to continue. Enter a name for the query and click Finish to create your query, which appears in table form.

Create a New Query Object
Step 1: Open Design View

In the New Query dialog box, select Design View and click OK. This opens the Show Table dialog box.
Step 2: The Show Table Dialog Box

Choose a table you want to use and click Add. Repeat this step to add tables to the query. Click Close when you're finished.
Step 3: Specify Which Table

Access displays the query in Design view. Each column in the grid at the bottom of the window represents a field you can use in your query. You can start by designating a table; click in the Table row of the first blank column, and then use the drop-down arrow to open the list and select a table for your query.
Step 4: Specify a Field

Click the Field row, and then use the drop-down arrow to choose a field.
Step 5: Enter the Criteria

The Criteria row lets you include only the records that match your specifications and your criteria. Click the Criteria row in the field you want to establish criteria for, and enter the criteria you want to use. You can use mathematical operators and comparison operators, such as greater than (>), less than (<), and equal to (=), in your criteria.
Step 6: Run the Query

Continue entering additional columns containing query fields, tables, and criteria. When you're ready to run the query, click the Run button on the toolbar and the results appear in a separate datasheet.

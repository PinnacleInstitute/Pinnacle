Backup a Table
Step 1: Select the Table

If you want to make a backup copy of a table, open the database that contains the table you want to copy. Click the Table button on the Objects bar in the Database window and select the appropriate table.
Step 2: Copy the Table

With the table selected, choose Edit and select Copy or click the Copy button on the toolbar. A copy of the table is placed on the Windows Clipboard.
Step 3: Paste the Table

To paste the table, choose Edit and select Paste or click the Paste button on the toolbar. Access displays the Paste Table As dialog box.
Step 4: Name the Duplicate Table

Type a name for the Duplicate table in the Table Name field of the dialog box. You might want to keep the same name for the table but add the text backup to the file name. To make a backup of both the table structure and the data in the table, choose the Structure and Data option. The other options only copy the structure or the data but not both. After clicking the OK button, the new table will appear in the list of tables.

Document a Table
Step 1: Access Documenter

When you work in the Design view, you can move from field to field to see the properties of each field in the table. By using the Access Documenter tool, you can view and print all the specifications for each field in a table. This tool is useful for creating a hard copy of your database design. To open the documenter, choose Tools, Analyze, and Documenter.
Step 2: Select the Table to Document

Open the Tables tab and select the table you want to document. Then click the Options button to open the Print Table Definition dialog box. Choose the option settings you want for the documentation. For example, you can choose field names, data types, sizes, and properties.
Step 3: View the Report

Click OK to close the Options dialog box, and then click OK in the Documenter dialog box. The Object Definition report displays, as shown in this screen. You can use the scroll bars to view the entire report.
Step 4: Print the Report

Click the Print button on the Object Definition Report toolbar to print a hard copy of the report. It's a good idea to keep the printed copy of the report for a reference in case you need to understand how a table is structured. Click the Close button on the Object Definition window when you are finished viewing and printing the report.

Sort Records in a Table
Step 1: Open the Table

Click the Tables object in the Objects bar of the Database window. The tables in the current database are listed, as shown in this figure. Either double click the desired table, or click the table to select it and then click the Open button. Both methods open a table.
Step 2: Select the Field to Sort

After you have created a table, you might want to view data that is sorted by state, zip code, or last name. Start by selecting the field that you want to sort by. For example, if you want to sort by last name, you would select the last name field. Click the column header Customer Name to select the entire column of data, or you can simply click in any record in the field you want to sort.
Step 3: Sort in Ascending Order

Click the Sort Ascending button on the toolbar or choose Records, Sort, and Sort Ascending. All the records in the table will be sorted on the designated field alphabetically from A to Z or numerically from lowest to highest.
Step 4: Click the Sort Descending button

Click the Sort Descending button on the toolbar or choose Records, Sort, and Sort Descending. All the records in the table will be sorted on the designated field alphabetically from Z to A or numerically from highest to lowest. Remember that sorting does not affect the order of records in the table because the primary key controls the actual order of records. Sorting only changes the view of the records.

Move Data to Another Table
Step 1: Open the Table

Click the Tables object in the Objects bar of the Database window. The tables in the current database are listed, as shown in this figure. Either double click the desired table, or click the table to select it and then click the Open button. Both methods open a table.
Step 2: Select the Records to Move

To keep a database from becoming too large with out-of-date data, such as former employees, you will want to remove those records. However, if you simply delete the records, you will loose all of the information associated with that record. Instead, you should move the records to another table, such as a Former Employee table. Select the record to move by clicking the row indicator to the left of the record.
Step 3: Use the Cut Command

With the record or records selected, choose Edit and select Cut or click the Cut button on the toolbar. The record is removed and is placed on the Clipboard.
Step 4: Paste the Record

Open or switch to the table to receive the record. In order to move records to another table, the second table must contain the same field names and they must appear in the same order and use the same data type. Otherwise, Access will not paste the data in the new table. Choose Edit, Paste Append, and Access adds the record to the bottom of the new table.

To create a query you need to first determine what question you want to ask. Click Queries in the list of objects.
Step 2: Use the Wizard

Double-click Create query by using wizard.
Step 3: Select Calls Fields

The Calls table appears in the Tables/Queries text box. You will want to add all six of these fields to your query so click the Select all Fields button as shown.
Step 4: Select Contacts Fields

Click the Tables/Queries list arrow, then click Table: Contacts. You only need to include the Location field in this table, so click the Contact Types field.
Step 5: Name the Query

Click Next to go to the next dialog box in the Query wizard where you can enter a name for the query and decide whether to view the query as it stands now, or modify the query design. Click the option button next to Modify the query design so you can enter specific criteria to a question, such as: Which performers will perform on June 21 at the Lakeview Arena?
Step 6: View the Query Design

Click Finish to view the current design of the query. If you ran the query now, all the performances would be listed in date order from June 20 to June 22. However, you want to enter specific criteria for your query-which you'll do next.

Step 1: Export to Word

You can also print the Access report as a Word document. This feature comes in handy when you want to quickly include Access data in a report or other kind of document that you've formatted in Word. Click the Office Links button and then click Publish It with Microsoft Word.
Step 2: Select Text in Word

Now that you have the report displayed in Word, you can select all the text at once by pressing Ctrl+A.
Step 3: Convert to a Table

The fastest way to format the Access data in Word is to convert the data to a Word table and then apply one of Word's preset table AutoFormats. First, click Table on the menu bar, point to Convert, and then click Text to Table.

Click OK to accept 4 as the number of table columns.

Find and Replace Options
Step 1: Open the Table

Click the Tables object in the Objects bar of the Database window. The tables in the current database are listed, as shown in this figure. Either double click the desired table, or click the table to select it and then click the Open button. Both methods open a table.
Step 2: Open Find or Replace

To open the Find and Replace dialog box, click the Find button on the toolbar or choose Edit, Find or Edit, Replace. After typing the data to find or find and replace, click the More button in the dialog box.
Step 3: Set Search Options

You can set options to help narrow the search. Use the Look in option to change the field or table to search. Use the Match option to have Access match the entire field, match any part of a field, or match only the beginning of a field. You can use the Match Case option to limit the search for words with the same capitalization or to expand the search to find all instances regardless of the capitalization. Use the Search Fields as Formatted option to find only matches to the formatting you specify.
Step 4: Find or Replace Data

After specifying Find and Replace options, click Find Next. If you have problems with the search, check that you specified all options correctly.

Check Your Spelling
Step 1: Open or Select a Table

Click the Tables object in the Objects bar of the Database window. The tables in the current database are listed, as shown in this figure. Either click the desired table, or click the table to select it and then click the Open button.
Step 2: Start the Spell Checker

With the table you want to check selected in the Database window or opened on the screen, choose Tools and select Spelling. The Spelling dialog box opens.
Step 3: Ignore Errors

The Spell Checker flags words that don't appear in its dictionary; therefore, many words in a database can appear as errors. You can ignore the current selection by clicking the Ignore button. You can ignore the selection in the entire table by clicking Ignore All, or if you want Access to skip a field entirely, such as a last name field, you can click the appropriate button. You can add the selection to the dictionary by clicking the Add button.
Step 4: Change Errors

If you want Access to change the current selection, click the Change button, and if you want Access to change the selection throughout the entire table, click the Change All button. You can stop the spell check at any time by clicking the Cancel button.

Data Entry Shortcuts
Step 1: Open the Table

Click the Tables object in the Objects bar of the Database window. The tables in the current database are listed, as shown in this figure. Either double click the desired table, or click the table to select it and then click the Open button. Both methods open a table.
Step 2: Repeat Data

If you have entered data in the previous record and you want to repeat that data in the same field in the current record, you can use a shortcut from the keyboard to do so. Place the cursor in the desired field and press Ctrl+', that is Ctrl+ the single quotation mark. The data from the previous record will be repeated in the field.
Step 3: Enter Date and Time

If you want to enter the date or time into a field and want to add it quickly and easily, you can use a shortcut from the keyboard to do so. Place the cursor in the desired field and press Ctrl+;, that is, Ctrl+ the semicolon, for the date, or Ctrl+:, that is Ctrl+ the colon for the time.
Step 4: Insert Record Contents

If you left data out of a field, for example, you forgot to input the street name in an address field, you can easily insert the data. Click the mouse in the field at the desired location, and simply type the data to be inserted. Existing data in that field will move to the right to make room for the inserted data.
Step 5: Save Changes

Access automatically saves changes to the table data as you input, so you will not be prompted to save the table when you close it. The next time you open the table, it will reflect the changes you made to the record data.

Make the Transition to Excel 10
Step 1: New Features

Excel now supports Extensible Markup Language (XML). Users can load and save generic, well formed, and spreadsheet XML directly into Excel. Furthermore, users can query XML spreadsheets from the Web and thereby analyze important Web data.
Step 2: Office Web Components

A variety of improvements have been made to the Excel Web Components, such as support for loading native Excel files, support for named ranges, multiple worksheets, wrapped texts, and the ability to publish entire workbooks with interactivity to the Web. The Excel PivotTable® component offers conditional filtering on a range of expressions for filtering data. Finally, Office XP extends the object model for the components that allow developers to build custom solutions that deliver data to the browser.
Step 3: Smart Tags

Microsoft Office Smart Tags are a set of buttons that are shared across the Office applications. These buttons appear when the user needs them (such as when a user makes an error in an Excel formula, when Word automatically corrects a user’s action, or when a user pastes some data) and gives the user the options they need to change the given action or error.
Step 4: Paste Options

Multiple paste options allow users to decide whether they want to paste their data as they originally copied it, change the style so that it fits the style of the document they are pasting the data into, or apply specific characteristics to the data based on the content.

Start Excel
Step 1: Start Excel

Open Excel XP if it is not already open. To do this, go to the Start menu and select Programs, and then Microsoft Excel.
Step 2: New Document

When you first start Excel 10, it automatically opens a new document. Go ahead and enter some data, so you'll have something to work with for this tutorial.
Step 3: The Toolbar

The toolbar contains some useful buttons, and in the next few steps you'll be using the New, Open, Save, and Print buttons. Take a minute to find them on the toolbar. Depending on how your version of Excel XP is set up, you may find your buttons move around the toolbars a bit. This is because Excel keeps track of the buttons you use most and moves them to the left of the toolbar. However, if you hold your mouse over a button, a tool tip displaying its name will appear.
Step 4: Save

Save your file by clicking the Save button (or selecting Save from the File menu). The Save As dialog box appears. The drop-down box labeled "Save in" shows the folder in which Excel is suggesting you save your file (usually the My Documents folder). The drop-down box labeled "File name" shows the name Excel is suggesting you use to call your file- by default it shows the first few words you typed in the document. Click Save to save your file.

Start a New Workbook
Step 1: Start Excel

Open Excel XP if it is not already open. To do this, go to the Start menu, select Programs and then Microsoft Excel.
Step 2: Select from the Menu

When you first launch Excel, a blank workbook will appear, ready for you to begin entering data. If you don’t see a blank, workbook, choose File and select New from the menu bar to start a new document. The New dialog box will appear. Click on the General tab and double-click on the Blank document icon. A blank workbook will appear.
Step 3: Look Around

Look around the screen to get your bearings. Pause the mouse for a few seconds over each button that appears on the top rows of the screen, or the toolbar. A tip will appear showing what each button does.
Step 4: Start Entering Data

Begin entering your data. When you reach the end of a cell, the text will automatically wrap around to the next line. Press the enter key only when you have finished entering data in that cell.

Save Your Workbook
Step 1: Save for the First Time

Click the Save button on the toolbar. Click the pop-up menu that appears and select the folder where you want to save the file. Enter a name for the file and then click Save.
Step 2: Save a File With a New Name

Click the File menu and select Save As. Enter a new name in the Save dialog box and click Save.
Step 3: Save a File in a New Place

Click the File menu and select Save As. Click the pop-up menu and then select the folder where you want to save the file. Click Save.
Step 4: Save a File in a New Format

Click the File menu and select Save As. Click the Save File As Type pop-up menu and select the file type you want. Click Save.

Enter Data
Step 1: Enter a Text Label

Click the cell where you want to enter a label. Type a label. A label can include upper and lowercase letters, spaces, punctuation and numbers.
Step 2: Enter a Number

Click the cell where you want to enter a number as a label. Type an apostrophe. The apostrophe is a label prefix and does not appear in a worksheet. Enter a number value. Examples of numbers that you might use as labels include a year, a social security number or a telephone number. Click the Enter button on the formula bar.
Step 3: Enter a Value

Click the cell where you want to enter a value. Type a value. To simplify your data entry, type the values without commas and dollar signs and apply a numeric format to them later. Click the Enter button on the formula bar.
Step 4: AutoComplete

To enter a Text Label with AutoComplete begin typing in a cell. If a previous entry in that column begins with the same characters, AutoComplete displays the entry. Press Enter to accept the entry, or resume typing to ignore the AutoComplete suggestion, and then click the Enter button on the formula bar.

Edit Data
Step 1: Select Cell

Double click the cell you want to edit. The insertion point appears within the cell. The Status bar will now display Edit rather than ready.
Step 2: Position in the Cell

Use the mouse pointer or the Home, End and arrow keys to position the insertion point in the cell.
Step 3: Erase

Use the Delete key to erase unwanted characters.
Step 4: Accept

Click the Enter button on the formula bar to accept the edit, or click the Cancel button to cancel it.

Basic Formulas
Step 1: Enter a Formula

Click the cell where you want to enter a formula. Type an equal sign to begin the formula. If you don’t do this, Excel will treat it as text and not a formula.
Step 2: Enter Argument

Enter the first argument. An argument can be a number or a cell address. If it’s a cell address, you can type the address or click the cell in the worksheet. Enter an arithmetic operator. Now, enter the next argument and click the Enter button on the formula bar.
Step 3: Copy a Formula

Select the cell that contains the formula you want to copy. Point to the fill handle located in the lower right corner of the selected cell. The pointer will change to a black plus sign.
Step 4: Display Formulas

Click the Tools menu and select Options. Click the View tab and select the Formulas check box. Click OK when you’re finished.

Format Text & Numbers
Step 1: Format Text

Select a cell or range that contains the text you want to format. Click the Bold, Italic, or Underline button on the Formatting toolbar to apply the style you selected to that range.
Step 2: Use the Format Cells Dialog Box

Select a cell or range that contains the text you want to format. Click the Format menu, and then select Cells. Click the Font tab and click to select the font, font style and size you would like. Preview your selections and then click OK.
Step 3: Format Numbers

Select the cell or range that contains the numbers that you want to format. Click the Currency Style, Percent Style, Comma Style, Increase Decimal, or Decrease Decimal button on the Formatting toolbar to apply the numeric attribute you want to the selected range.
Step 4: Use the Format Cells Dialog Box

Select a cell or range that contains the numbers you want to format. Click the Format menu and the select Cells. Click the Number tab and click to select a category. Preview your selections and then click OK.

Sort Your Data
Step 1: Select Range

Click anywhere within the list range. Click the Data menu and then select Sort.
Step 2: Sort By

Click the Sort By pop-up menu and then select the field on which the sort will be based.
Step 3: Ascending or Descending

Click the first Then By drop-down list, select a secondary sort field and then click the Ascending or Descending option button.
Step 4: Exclude

Click the Header Row option button to exclude the field names from the sort, or click the No Header Row option button to include the field names in the sort. Click OK when you’re finished.

Find and Replace Data
Step 1: Use the Find Command

To search for a word or number in a worksheet, open the Edit menu and select Find.
Step 2: Enter the Search Data

In the Find and Replace dialog box, click inside the Find What text box and type the characters (numbers, text, symbols) for which you want to search.
Step 3: Select What to Search

Click the Look In drop-down arrow and select what you want to search. For example, if you're searching for text, select Values; if you're searching formulas, select Formulas; and if you're searching cell comments, select Comments.
Step 4: Start the Search

Click the Find Next button to start the search. Excel locates the first cell containing your search characters and highlights it in the worksheet. Click Find Next repeatedly to find each occurrence of the character string.
Step 5: Use the Replace Command

To conduct a search and replace, you must first open the Replace dialog box. Open the Edit menu and select Replace.
Step 6: Replace the Data

In the Replace dialog box, type the data you want to replace in the Find what box. Type the replacement data in the Replace with box. Click Find Next, and Excel highlights the first occurrence in the worksheet. Click the Replace button to replace the data; click Replace All to replace all occurrences at once.

Basic Charting
Step 1: Select the Range

Select the data or range you want to chart. Click the Chart Wizard button on the toolbar.
Step 2: Select Chart Type

Click a chart type and sub-type. Click the Press and Hold To View Sample button to preview your selection, and then click Next.
Step 3: Rows or Columns

Select the appropriate option button to plot the data series in rows or in columns, and then click Next.
Step 4: Enter Titles

To identify the data in the chart, type the titles in the appropriate text boxes and click Next. Select the option you want to place the chart on. This may be a new sheet or on an existing sheet. Click Finish.

Enhance Your Chart
Step 1: Add a Title

Select a chart to which you want to add a title. Click the Chart menu and select Chart Options. Click the Titles tab and enter the title of the chart. Press Tab and enter a title for the x-axis. Then press Tab again and enter a title for the y-axis. Click OK.
Step 2: Add or Delete a Legend

Select the chart to which you want to add or delete a legend. Click the Chart menu and then select Chart Options. Click the Legend tab and click the Show Legend check box to add or delete a legend. Click a Placement option to indicate the location of the legend and click OK.
Step 3: Format a Chart Element

Select a chart to which you want to format an element. Double click the chart element you want to format. Click the appropriate tab with the type of change you want to make. Click the formatting options you want to change. Click OK when you’re finished.
Step 4: Add Gridlines

Select a chart to which you want to add gridlines. Click the Chart menu and then select Chart Options. Click the Gridlines tab. Select the type of gridlines you want for the x-axis and y-axis. Click OK when you’re finished.

Edit Cell Contents
Step 1: Select Cell

Double click the cell you want to edit. The insertion point appears within the cell. The Status bar will now display Edit rather than ready.
Step 2: Position in the Cell

Use the mouse pointer, or the Home, End and, arrow keys to position the insertion point in the cell.
Step 3: Erase

Use the Delete key to erase unwanted characters. Type new characters as needed.Accept
Step 4: Accept

Click the Enter button on the formula bar to accept the edit, or click the Cancel button to cancel it.

Work With Cells
Step 1: Insert Cells

Select the cell or cells where you want to insert the new cells. Click the Insert menu and select Cells.
Step 2: Options, Options

Select the option you want. If you want the contents of the cells to move right, click the Shift Cells Right button. If you want the contents of the cells to move down, click the Shift Cells Down option button. Click OK when you’re finished.
Step 3: Delete Cells

Select the cell or cells you want to delete. Click the Edit menu and select Delete.
Step 4: Options, Options.

Select the option you want. If you want the remaining cells to move left, click the Shift Cells Left button. If you want the remaining cells to move up, click the Shift Cells Up button. Click OK when you're finished.

Select Multiple Cells
Step 1: Select a Range

Click in the first cell you want to include in the range. Drag the mouse diagonally to the last cell you want to include in the range. When a range is selected, the active cell border surrounds the top leftmost cell, while the additional cells are highlighted in black.
Step 2: Non-contiguous Range

Click the first cell you want to include in the range. Drag the mouse diagonally to the last contiguous cell. Press and hold the Command key and drag over the next group of cells you want in the range. Repeat this until all of your cells are selected.
Step 3: Name a Range

Select a range you want to name. Click the Name box on the formula bar. Enter a name of the range. Press Enter and the range name will appear.
Step 4: Select a Named Range

Click the Name box drop down arrow on the formula bar. Click the name of the range you want to select.

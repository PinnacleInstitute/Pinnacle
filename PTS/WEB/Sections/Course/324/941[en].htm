Move & Copy Worksheets
Step 1: Move a Sheet

Click the sheet tab of the worksheet you want to move and then press and hold the mouse button. The pointer will change to a small sheet.
Step 2: Select Location

Drag the pointer/sheet to the right or in front of the sheet tab where you want to move the worksheet.
Step 3: Copy a Sheet

Click the sheet tab of the worksheet you want to copy. Click the Edit menu and select Move or Copy Sheet. If you want to copy the sheet to another workbook, click the To Book pop-up menu and select that workbook. The sheets of the selected workbook appear in the Before Sheet Box.
Step 4: Click a Sheet Name

Click a sheet name in the Before Sheet list box. The copy will be inserted before this sheet. Click the Create a Copy check box. Click OK.

Insert a Column or Row
Step 1: Insert a Column

To insert a column click anywhere in the column to the right of the location of the new column you want to insert. Select Insert Column from the Edit Menu.
Step 2: Insert a Row

To insert a row, click anywhere in the row immediately below the location of the row you want to insert. Select Insert and click Rows.
Step 3: Insert Multiple Columns

To add multiple columns, drag to select the column indicator buttons for the number of columns to be inserted. Select Insert and click Columns.
Step 4: Insert Multiple Rows

To add multiple rows, drag to select the row indicator buttons or the number of rows to be inserted. Select Insert and click Rows.

Rearrange Worksheet Data 1
Step 1: Select Cells to Move

To move cells, select the range of cells you want to move, and point to any border of the selected range so that the mouse pointer becomes an arrow. Drag the range to a new location.
Step 2: Drop the Cells

While you drag, an outline of the range moves across the worksheet, and a ScreenTip tells you the reference of the range location. When the range border is where you want it, drop the data by releasing the mouse button.
Step 3: The Range Moves

The range moves to its new location.
Step 4: Select Cells to Copy

To copy cells, select the range of cells you want to copy, and point to any border of the selected range so that the mouse pointer becomes an arrow. Drag the range to a new location.
Step 5: Drag the Cells

While you drag, an outline of the range moves across the worksheet, and a ScreenTip tells you the reference of the range location.
Step 6: Press Ctrl and Drop the Cells

When the range border is where you want it, press and hold Ctrl, and then drop the data by releasing the mouse button. When you press Ctrl, the mouse pointer acquires a small plus symbol that tells you it's copying. A copy of the data is dropped in the new location.

Rearrange Worksheet Data 2
Step 1: Select the Data to Move

To move data, you will use the Cut and Paste commands. Select the data you want to move. Right-click the selected cells.
Step 2: Cut the Selected Data

Choose the Cut command on the shortcut menu. The selected data will have a moving border around it that disappears when you paste it somewhere else.
Step 3: Paste the Cut Data

Right-click the upper-left cell of the range where you want to paste the data, and choose Paste on the shortcut menu. The data is pasted into its new location and disappears from its original location.
Step 4: Select the Data to Copy

To copy data, you will use the Copy and Paste commands. Select the data you want to copy. Right-click the selected cells.
Step 5: Copy the Selected Data

Choose the Copy command on the shortcut menu. The selected data will have a moving border around it.
Step 6: Paste the Copied Data

Right-click the upper-left cell of the range where you want to paste the copied data, and choose Paste on the shortcut menu. The data is pasted into its new location (the moving border remains around the copied data until you begin typing in another cell, or until you press Enter or Esc).

Select a Range of Cells
Step 1: Click the First Cell

To select a range, start by clicking the first cell in the range.
Step 2: Drag to Select

Hold down the mouse button and drag across the cells you want to include in the range.
Step 3: Release the Mouse Button

When you release the mouse button, the range is selected. Ranges are referred to by their anchor points, the upper-left corner and the lower-right corner. After selecting a range, you can format the cells using Excel's formatting features, or use the range in a formula.
Step 4: Select a Range with the Keyboard

To select a range using the keyboard, use the arrow keys to move to the first cell in the range.
Step 5: Use the Shift Key

Press and hold the Shift key, and then use the arrow keys to select the range.

Copy Formatting
Step 1: Select the Range

Select the range that has the formatting you want to copy.
Step 2: Choose Format Painter

Click the Format Painter button in the Standard toolbar.
Step 3: Mouse Pointer Changes

Your mouse pointer changes to a paintbrush pointer.
Step 4: Drag to Copy Formatting

Drag the paintbrush pointer across the range where you want to paint the format. Release the mouse. The formatting is painted to the range of cells (click anywhere to deselect the range).

Select a Range of Cells
Step 1: Click the First Cell

To select a range, start by clicking the first cell in the range.
Step 2: Drag to Select

Hold down the mouse button and drag across the cells you want to include in the range.
Step 3: Release the Mouse Button

When you release the mouse button, the range is selected. Ranges are referred to by their anchor points, the top left corner and the bottom right corner. After selecting a range, you can format the cells using Excel's formatting features, or you can use the range in a formula.
Step 4: Select a Range with the Keyboard

To select a range using the keyboard, use the arrow keys to move to the first cell in the range.
Step 5: Use the Shift Key

Press and hold the Shift key and use the arrow keys to select the range.
Step 6: Select the Entire Worksheet

To select the entire worksheet as a range, click the Select All button located at the top of the worksheet to the left of the column labels.

Delete Cells
Step 1: Select the Cells

Select the cells you want to delete.
Step 2: Choose Delete

Right-click the selected cells and choose Delete from the shortcut menu.
Step 3: Shift Replacement Cells Up

In the Delete dialog box, click the Shift Cells Up option and click OK.
Step 4: Replacement Cells Move

The cells are removed from the worksheet, and the cells below move up to fill in the empty space. The cells on either side of the deleted cells are unaffected.
Step 5: Shift Cells Left

If you select the Shift Cells Left option in the Delete dialog box (in step 3), cells on the right side of the deleted cells move over, and cells above and below the deleted cells are unaffected.
Step 6: Clear the Formatting Only

If you don't want to delete the cells or the contents, but you want to remove their formatting (cell borders and colors) without affecting the data, select the cells and choose Edit, Clear, and Formats.

Delete Columns and Rows
Step 1: Select the Row

To delete a row, select the row you want to delete (click its row number to select it).
Step 2: Delete the Selected Row

Right-click the selection and select Delete on the shortcut menu.
Step 3: The Row Is Removed

Excel immediately deletes the entire row, and the existing rows move up to fill its place.
Step 4: Select the Column

To delete a column, select the column you want to delete (click its column letter to select it).
Step 5: Deleted the Selected Column

Right-click the selected column and select Delete on the shortcut menu. As soon as you select the command, the column is immediately deleted.
Step 6: Delete Multiple Rows

To delete several adjacent rows all at the same time, select the adjacent rows in the worksheet, right-click the selection, and choose Delete from the shortcut menu.
Step 7: Delete Multiple Columns

To delete several adjacent columns at the same time, select the adjacent columns in the table, right-click the selection, and choose Delete from the shortcut menu.

Create a List
Step 1: Open a Blank Worksheet

Open a blank worksheet or use a worksheet that has enough empty columns and rows for the list.
Step 2: Enter a Name

Enter a name for each field in adjacent columns across the first row of the list.
Step 3: Enter Field Information

Enter the field information for each record in a separate row, starting with the row directly beneath the field names.
Step 4: Enter a Label

Control click the cell where you want to enter a label and then select Pick From List. Select an entry from the pop-up menu.

Sort Data in a List
Step 1: Select Range

Click anywhere within the list range. Click the Data menu and then select Sort.
Step 2: Sort By

Click the Sort By pop-up menu and then select the field on which the soft will be based.
Step 3: Ascending or Descending

Click the first Then By pop-up menu, select a secondary sort field and then click the Ascending or Descending option button.
Step 4: Exclude

Click the No Header Row option button to exclude the field names from the sort, or click the Header Row option button to include the field names in the sort. Click OK when you’re finished.

Spell Check Your Work
Step 1: Open Spelling & Grammar

Click at the beginning of the document and then click the Spelling and Grammar button on the toolbar. As it checks each sentence in the document, Excel highlights misspelled words in the Not in Dictionary box or problematic sentences in the Sentence box and displays an appropriate alternative in the Suggestions box.
Step 2: Make a Substitution

To make a substitution, select any of the suggestions and then click change.
Step 3: Skip the Word

To skip a word or rule, click Ignore Once or to skip every instance of the word or rule, click Ignore all.
Step 4: Do it Yourself

If none of Excel’s suggestions seem appropriate, click in the document and edit yourself. Click the Resume button to continue.

Undo Problems
Step 1: Undo

Click the Undo button to reverse your most recent action, such as typing a word or deleting something.
Step 2: Redo

Click the Redo button to restore the last action you reversed.
Step 3: Undo Drop Down Menu

Use the Undo drop down menu and then click the actions you want to reverse.
Step 4: Redo Drop Down Menu

Use the Redo drop down menu and then click the actions you want to restore.

Preview and Print
Step 1: Select Print

Click the File menu and select print. Click the Printer pop-up menu and select the printer you want.
Step 2: How Many?

Specify the pages you want to print and how many copies you would like.
Step 3: Preview

Click the Preview button at the bottom of the dialog box to see your page as it will be printed.
Step 4: Print

When you’re satisfied with your selections, click OK.

Close Your Work
Step 1: Use the File Menu

Click the File menu and then select Close.
Step 2: Use the Close Box

Click the Close box on the title bar.
Step 3: Save and Close

Click the File menu and then select Save. Save the file and then close it.
Step 4: Exit

Click the File menu and select Exit.

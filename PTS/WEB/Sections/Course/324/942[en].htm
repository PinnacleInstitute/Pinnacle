Format a Chart
Step 1: Change the Font

To change a font for any chart text, first select the text, then open the Text menu and choose Font. This opens the Font dialog box.
Step 2: Select Another Font

Choose the font you want to use, and change the font style or size if needed. Click OK to close the dialog box and apply the changes.
Step 3: Change a Box Border

To change a border for any chart box, first select the box, then open the Boxes menu, choose Border Styles and then select the style you want to apply.
Step 4: Close the Chart

When you're finished constructing and formatting your chart, simply click the OK button in the Format Chart Area box.

Use Fonts
Step 1: The Font Combo Box

The first item on the formatting toolbar is the Font combo box. Click on the box’s drop-down arrow to see a list of available Fonts.
Step 2: Select Text

Select the text in your workbook that you would like to apply a different font to by dragging the mouse to the left across the text while holding down the right mouse button.
Step 3: Select a Font

Using the Font combo box, pull down the font list and select a new font.
Step 4: New Font

Your text should now be in the font that you selected. If you don’t like the font you’ve selected, simply repeat the above steps and select a new font.

Use the Font Style Buttons
Step 1: Click the Appropriate Button

Once you have entered your data, you will want to give emphasis to particular items by using the bold, italic or underline formatting features. Select the text you want to format, or choose the formatting commands before typing the text. To boldface text, click the Bold button on the Formatting toolbar. To italicize text, click the Italic button. To underline text, click the Underline button.
Step 2: Formatting Applied

Depending on which buttons you select, the formatting is immediately applied to your selected text.
Step 3: Open the Font Dialog Box

If you want to see what the formatting will look like before you apply it, open the Format menu and choose Cells. Click on the Font tab in the Format Cells dialog box.
Step 4: Use the Font Dialog Box

In the Font style list, choose Bold, Italic, or Bold Italic to boldface or italicize your text. To underline text, click the down arrow on the Underline list box to display the list of choices, and then click the desired underline style.
Step 5: Preview Your Selections

In the Preview area at the bottom of the dialog box, you can preview how your choices will affect the text.
Step 6: Exit the Dialog Box

When you've made your selections, choose OK to close the dialog box and apply the changes to the selected text.

Align Data
Step 1: Select Location

Select a cell or range that contains the data you want to realign. Click the Format menu and select Cells.
Step 2: Alignment

Click the Alignment tab. Click the Horizontal drop down list and then select an alignment.
Step 3: Orientation

Select an orientation by clicking on a point on the orientation map or by clicking the Degrees up or down arrow.
Step 4: Text Control

If necessary, click one or more of the Text Control boxes. Click OK when you’re finished.

Number Style Format
Step 1: Select the Cell or Range

Select the cell or range where you want to change the number format.
Step 2: Click Currency

To change a format to accounting format (which adds a $, rounds the number to two decimal places, and spaces the $ so that all the dollar signs in the column are aligned), click the Currency Style button on the Formatting toolbar.
Step 3: Click Percent

To change a format to percent format (which adds a % and changes the number from a fraction or integer to a percentage value), click the Percent Style button on the Formatting toolbar. (Remember, percent means hundredths, so 0.12 is displayed as 12%, but 12 will be displayed as 1200%).
Step 4: Click Comma

To apply comma format (which rounds the number to two decimal places and adds a comma at each thousands mark), click the Comma Style button on the Formatting toolbar. To control the decimal point, you can also click the Increase Decimal or Decrease Decimal buttons on the Formatting toolbar.
Step 5: Use the Format Cells Dialog Box

To apply different formats that aren't on the Formatting toolbar, open the Format menu and select Cells. This opens the Format Cells dialog box. Click the Number tab and then choose a number format in the Category list. Your selected number is displayed with that format in the Sample area.
Step 6: Set Format Options

Different formats in the Format Cells dialog box offer different options; select a Category and set the options for that format. A description of the selected category appears at the bottom of the dialog box. To exit the dialog box and apply the new format, click OK.

Add Borders to Cells
Step 1: Select Cells

Select a cell or range where you want to apply borders, or to select the entire worksheet, click the Select All button.
Step 2: Open Format Cells

Click the Format menu and then select Cells. Click the Border tab and select a line type from the Style box.
Step 3: Inside or Outside?

If you want a border on the outside of a cell or cell range, or lines inside a range of cells, click the Outline or Inside icon. If you want to remove a border, click the None icon.
Step 4: Other Options

To choose the other available Border options, click a Border icon, or click in the Border box at the location where you want the border to appear. Click OK when you’re finished.

Add Patterns
Step 1: Open the Format Cells Dialog Box

Select the cell or range to which you want to add a border or pattern, open the Format menu, and select Cells.
Step 2: Use the Patterns Tab

To apply a pattern, click the Patterns tab. To select a color, click the color effect you want to apply from the palette. Click the Pattern drop down list to select a pattern to apply. Click OK to exit the dialog box and apply the pattern or shading.
Step 3: Click OK

Click OK to exit the dialog box and apply the pattern or shading.
Step 4: The New Formatting Is Applied

As soon as you exit the Format Cells dialog box, the new settings are applied to your worksheet, as shown here.

Format Text & Numbers
Step 1: Format Text

Select a cell or range that contains the text you want to format. Click the Bold, Italic or Underline button on the Formatting toolbar to apply the style you selected to that range.
Step 2: Use the Format Cells Dialog Box

Select a cell or range that contains the text you want to format. Click the Format menu, and then select Cells. Click the Font tab and click to select the font, font style, and size you would like. Preview your selections and then click OK.
Step 3: Format Numbers

Select the cell or range that contains the numbers that you want to format. Click the Currency Style, Percent Style, Comma Style, Increase Decimal, or Decrease Decimal button on the Formatting toolbar to apply the numeric attribute you want to the selected range.
Step 4: Use the Format Cells Dialog Box

Select a cell or range that contains the numbers you want to format. Click the Format menu and the select Cells. Click the Number tab and click to select a category. Preview your selections and then click OK.

Use AutoFill
Step 1: Start a Month List

To AutoFill a list of month names, enter a single month name in a cell, and then select that cell. You can enter an abbreviation or type in the entire month name.
Step 2: Drag the Fill Handle

Move the mouse pointer over the fill handle until it takes the shape of a crosshair. Next, click and hold down the left mouse button while you drag across a row or column of cells. A ScreenTip shows what's being filled into each cell you drag.
Step 3: Release the Fill Handle

Release the mouse button at the end of the row or column of cells you want to fill. The series is entered in the cells in the proper order.
Step 4: Start a Text Series

To enter a text series, such as Div 1, Div 2, and so on, click inside the cell and enter the first entry in the series. Select the cell and drag the fill handle in the direction you want to go.
Step 5: Release the Fill Handle

Release the fill handle and the text series appears in the selected cells.
Step 6: Start a Number Series

To fill a number series that increases by 5, start the series by entering 5 and 10, select the two cells, and drag the fill handle to fill the series. AutoFill automatically fills in the rest of the series based on the first two cell entries.

AutoFormat a Range
Step 1: Select the Range

Select the range containing the data you want to format.
Step 2: Open the AutoFormat Dialog Box

Open the Format menu and select AutoFormat. This opens the AutoFormat dialog box.
Step 3: Choose a Table Format

From the Table format list box, choose a format style you want to view. The Sample area shows a preview of the table format.
Step 4: View Your Options

To exclude certain elements from the format set, click the Options button.
Step 5: Turn Off Elements

Select or deselect the format options you want to turn on or off.
Step 6: Exit the Dialog Box

Click OK to exit the dialog box and apply the new formatting.

Rotate Text
Step 1: Select the Range

Select the cell or range of cells you wish to change the orientation of.
Step 2: Select Alignment Tab

Select Format and choose Cells from the menu bar. Click on the Alignment tab.
Step 3: Select Orientation

On the right side of the dialog box, either click on the vertical box to display your text top-to-bottom or click the right box for angular orientation.
Step 4: Select Degree of Grade

Click and hold on the word text in this box and drag it upward or downward to set the degree of the grade, or click on the Degrees box below this and manually enter a number.
Step 5: Click OK

Click the OK button. Your text should now appear to be printed on an angle.

Format Workbooks
Step 1: Three Worksheets

The Excel workbook contains three worksheets by default. The sheet name tabs are at the bottom of the Excel window.
Step 2: Show More Sheet Tabs

Place your pointer on the tab-split bar. When it is position correctly, the pointer becomes a double-headed arrow. Drag it to the right to show more sheet tabs.
Step 3: Show Fewer Sheet Tabs

Place your pointer on the tab-split bar. When it is positioned correctly, the pointer becomes a double-headed arrow. Drag it to the left to show fewer sheet tabs.
Step 4: Return to Default

To undo your adjustments, simply double click on the tab-split bar.

Arrange Multiple Workbooks
Step 1: Open Workbooks

If the sheets you need to work on are in different workbooks, follow these steps to see them all. First open all of the workbooks you need.
Step 2: Open Arrange Dialog Box

Select Window and choose Arrange from the menu bar to open the Arrange Windows dialog box.
Step 3: Select Option

Select one of the option buttons (Tiled, Horizontal, Vertical or Cascade) and click the OK button.
Step 4: Multiple Sheets in One Workbook

If the sheets you want to view are in the same workbook, select Window and choose New Window from the menu bar. This opens a new window with the same worksheet in it. Repeat this for each sheet you want to view. Once you have all of the windows that you need open, select Window and choose Arrange from the menu bar.

Hide and Unhide Columns
Step 1: Hide a Column

Select the column you want to hide by clicking on its column header.
Step 2: Right Click

Right click within the selected column and choose Hide from the shortcut menu that appears. The column should now be hidden. You can also accomplish this by selecting Format, Column, and Hide from the menu bar.
Step 3: Unhide a Column

Select a visible range of columns that includes the hidden column.
Step 4: Right Click

Right click within the selected column and choose Unhide from the shortcut menu that appears. The column should now be hidden. You can also accomplish this by selecting Format, Column, and Unhide from the menu bar.

Adjust Width and Height
Step 1: Drag a New Column Width

To adjust the column width, point at the right border of the column letter for the column you want to widen. The mouse pointer becomes a two-headed arrow. Hold down the left mouse button and drag the border in either direction to adjust the width. Release the mouse button when the column is the width you want.
Step 2: Resize Multiple Columns

To make several columns the same width, select all the columns by dragging over their column letters, and then adjust the width for any one of them while they're all selected. They will all adjust to the same width.
Step 3: Open the Column Width Dialog Box

To enter an exact value for the column width, select the column, open the Format menu, choose Column, and then Width.
Step 4: Specify an Exact Size

Enter a measurement (in points) for the column width, click OK to exit and apply the new setting.
Step 5: Drag a New Row Height

You can adjust row height the same way you adjust column width. Point to the bottom border of the row number for the row you want to deepen. The mouse pointer becomes a two-headed arrow. Hold down the left mouse button and drag the border in either direction to adjust the row height. Release the mouse button when the row is the height you want.
Step 6: Open the Row Height Dialog Box

You can also specify an exact value for the row height. Select the row or rows to which you want to apply a new measurement to. Open the Format menu, select Row, and then Height. Enter a new value (in points) and click OK to exit.

Move and Copy Text
Step 1: Select the Text

Select the text you want to cut or copy.
Step 2: Use Cut and Copy

An easy way to cut and copy is to use the toolbar buttons. To move the text, click the Cut button in the Standard toolbar. The text is deleted from your document, but it remains in a special Windows storage area called the Clipboard. To copy the text, click the Copy button in the Standard toolbar. When you copy text, nothing appears to happen because the text remains in its original location, but a copy of the selected text is sent to the Clipboard.
Step 3: Relocate the Cursor

Click to place the cursor in the document where you want to paste the cut or copied text. If necessary, you can open another document or switch to another already open document to paste text there.
Step 4: Use Paste

Click the Paste button in the Standard toolbar to paste the text. The text is pasted into the document beginning at the position of the insertion point.

Drag and Drop Method to Move Text
Step 1: Select the Text

Select the text you want to cut or copy.
Step 2: Drag and Drop to Move Text

Another easy method to move or copy text is to drag and drop it. To move text, click in the selected text and hold the mouse button down, then drag the mouse where you want to paste the text. Release the mouse button to drop or paste the cut text.
Step 3: Select the Text

Select the text you want to cut or copy.
Step 4: Drag and Drop to Copy Text

To copy text by dragging and dropping it, select the text you want to copy. Click in the selected text, hold down the mouse button and the Ctrl key, then drag the mouse to the new location. Release the mouse button to drop or paste the copied text.

Find and Replace
Step 1: Find a Word or Phrase

To perform a quick search of your document for a particular word or phrase, open the Edit menu and select Find. This opens the Find and Replace dialog box with the Find tab displayed.
Step 2: Enter the Text

Enter the text you want to search for in the Find what text box (for example, type using). If you want to specify search criteria, such as matching case or finding whole words only, click the More button to reveal search options you can choose.
Step 3: Conduct the Search

Click the Find Next button to locate the first occurrence of the word or phrase. Word highlights the text in your document, and the Find and Replace dialog box remains open on your screen. To search for the next occurrence, click Find Next again, or click Cancel to close the dialog box.
Step 4: Find and Replace Text

To find the text and replace it with new text, use the Replace command. Open the Edit menu and select Replace. This opens the Find and Replace dialog box with the Replace tab up front.
Step 5: Enter the Text

Type the word or words you're looking for in the Find what text box. Type the replacement text in the Replace with text box. (If you want to specify any search criteria, click the More button and select from the available options).
Step 6: Search and Replace

Click the Find Next button to locate the first occurrence. Word highlights the text in the document. Click the Replace button to replace the text with the new text. Click Replace All to replace every occurrence in the document. Click Find Next to ignore the first occurrence and move on to the next.

Use AutoCorrect
Step 1: Try it Out

Test AutoCorrect to see how it performs-type teh, then press the spacebar or type a punctuation mark such as a comma or a period. Because "teh" is a common misspelling, AutoCorrect corrects it to "the" before you realize you mistyped it.
Step 2: Undo AutoCorrect

If you type something you don't want corrected (for example, "Sigmund Teh"), press Ctrl+Z to undo the correction before you type any other characters. The AutoCorrection is undone, and you can continue typing.
Step 3: Remove a Word from AutoCorrect

To remove a word from the AutoCorrect list, choose Tools, AutoCorrect.
Step 4: Delete the Word

Click the AutoCorrect tab. In the Replace box, type the first few letters of the word you want to delete from AutoCorrect; the list of words and replacements scrolls to the point in the list where you can find your word. Click your word in the list, and then click Delete. Click OK to exit the dialog box.
Step 5: Add a Misspelling

To add a word you frequently misspell to the AutoCorrect list, open a document and type the correct spelling. (You can also add a long phrase you want to create a shortcut to; type in the phrase, including any special capitalization). Select the word, choose Tools and select AutoCorrect.
Step 6: Click Add

Click the AutoCorrect tab. Your word or phrase appears in the With box. In the Replace box, type the incorrect word (for example, signifigant) or acronym you want to replace (this text is what you will mistype in your document). If you only want to add a single word, choose OK to close the dialog box; if you want to add more words to the AutoCorrect list, click Add to add each word, and choose OK when you're finished.

Bookmarks
Step 1: Overview

Bookmarks allow you to mark places in a document. This can be useful when you are working in a long document or it can help you create links in a document using more sophisticated Word tools. This tutorial covers how to access the Bookmarks dialog and add a new bookmark to a document.
Step 2: Selecting a Spot

Before you can add a bookmark, you need to select the spot for the bookmark. Simply place your cursor next to the spot you wish to bookmark. For example, you may want to bookmark a chapter title.
Step 3: Accessing the Bookmarks Dialog

Click Insert and select Bookmarks in the menu bar. The Bookmarks dialog box opens. Note that on Word XP menus, this option may be hidden. To make it visible, click the down arrow at the bottom of the menu.
Step 4: Naming the Bookmark

Enter the name of the bookmark in the Bookmark Name box at the top of the dialog box. If you create a name with more than one word, you need to use underscores to separate the words, rather than spaces. For example, you would need to enter bookmark_one or the Add button dims and you cannot add the bookmark.
Step 5: Adding the Bookmark

With the bookmark named in the correct format, you can add it by clicking the Add button. This adds the bookmark to the document at the placement of your cursor.
Step 6: Making Bookmarks Visible

Once you add one or more bookmarks to a document, how do you know they are there? You need to turn on View/Bookmarks. Click Tools and select Options in the menu bar. Click the View tab and make sure Bookmarks is checked in the Show.

Organize Bookmarks
Step 1: Overview

You can organize the list of bookmarks in a number of ways including deleting a bookmark, displaying hidden bookmarks or sorting the bookmark list. Keep in mind, you should try to avoid renaming bookmarks. This is because if you have a reference to a particular bookmark name, and then change it, Word won't update the reference automatically and you will end up with an error in the document.
Step 2: Access the Bookmark Dialog

As you learned in the previous tutorial, you access the Bookmark dialog box by clicking Insert and selecting Bookmark in the menu bar. Remember, it may be hidden in which case; you need to click the down arrow at the bottom of the menu to make it visible.
Step 3: Sorting Bookmarks

You can sort your bookmarks by clicking one of the Sort Order option buttons at the bottom of the dialog. If you select By Name, Word sorts the bookmarks in alphabetical order by name. If you click By Location, Word sorts the bookmarks according to their location in the document beginning with the first bookmark.
Step 4: Deleting a Bookmark

If you have a long list of bookmarks, scroll through the list until you find the one you want. You can sort the bookmarks as described in the previous step. When you find the bookmark you wish to delete, click it, then click the Delete button. Word deletes the bookmark without further prompting.

Use Bookmarks
Step 1: Overview

With bookmarks in place, you can use the bookmarks to navigate within the document. This tutorial explains how to use the bookmarks to get around the document more easily. There are a couple of ways to do this including from the Bookmark dialog and the Go To tab in the Find and Replace dialog.
Step 2: Finding the Bookmark

As you learned earlier you can access the Bookmark dialog by selecting Insert and selecting Bookmark in the menu bar. When the Bookmark dialog appears, locate the bookmark you want.
Step 3: Moving to the Bookmark

Once you locate the Bookmark you want, click to select it, and then click the Go To button. The Bookmark dialog remains open, but the cursor moves to the placement of the bookmark. Click Close to close the Bookmark dialog.
Step 4: Go To Dialog

You can also navigate to a bookmark using the Go To tab in the Find and replace dialog box. Press Ctrl-G on your keyboard and the Find and Replace dialog opens with the Go To tab selected.
Step 5: Selecting Go To Type

In the Go To What column on the left, scroll down if needed, then click Bookmark.
Step 6: Selecting a Bookmark

Select a bookmark by entering the name in the Enter Book Name box or click the drop-down list and make a selection from the drop-down. Note that these are listed in alphabetical order and you cannot alter the sort order here as you can in the Bookmark dialog.
Step 7: Navigating to the Bookmark

Click the Bookmark you want, then click the Go To button at the bottom of the dialog. Word moves you to the placement of the Bookmark within the document. As with the bookmark dialog, it remains open. Click Close to dismiss the dialog.

Add to the Custom Dictionary
Step 1: Overview

Many words, such as proper names, business acronyms or legal terms, may not be in Word’s dictionary. Although you cannot make changes to Word’s dictionary, you can add to and edit words in the custom dictionary.
Step 2: Start Spell Checker

Start the spelling and grammar checker by pressing F7. Or you can select it from the menu bar by selecting Tools and choosing Spelling & Grammar.
Step 3: Click Add

When Word pauses on a word that is not in its dictionary that you think should be part of the custom dictionary, click on the Add button on the right side of the dialog box.
Step 4: Repeat

Repeat the previous step for as many words as you wish to add. If you inadvertently add a word, you can edit the custom dictionary later and remove the word.

Remove from the Custom Dictionary
Step 1: Open Options

Choose Tools and select Options from the menu bar. This will open the Options dialog box. Choose the Spelling & Grammar tab and click Dictionaries.
Step 2: Click Modify

When the dictionaries appear CUSTOM.DIC will be selected, because the custom dictionary is currently open in memory and being used in conjunction with Word’s dictionary. Click the Modify button at the top of the dialog box.
Step 3: Select Words

Word will now show the contents of the custom dictionary in a document. Scroll down and look at the words. Delete or edit the words that you do not wish to have in the dictionary just as you would any text in Word.
Step 4: Save the Custom Dictionary

Save the custom dictionary by clicking on the save button on the Standard toolbar or choose File and select Save from the menu bar. Select File and select Close to return to your document.

Create a New Dictionary
Step 1: Overview

You can create additional dictionaries to store words that you may want to group together, such as legal or computer terms. You create these new dictionaries for Word to reference when spell-checking your documents. Choose Tools and select Options to open the Options dialog box. Click on the Spelling and Grammar tab.
Step 2: Click Dictionaries

In the middle of the dialog box, to the right side, click on the button that says Dictionaries. The custom Dictionaries dialog box will now appear.
Step 3: Click New

Click on the New button. The Create Custom Dictionary dialog box will open, displaying the folder where all of the dictionaries are filed.
Step 4: Name Your Dictionary

Enter the name for your new dictionary and click Save. Word will add the .dic extension to your name so it knows that it is a dictionary file.

Use Templates
Step 1: Open the New Dialog Box

Open the File menu and select New to display the new dialog box. (You can't use the New button on the Standard toolbar as a shortcut. If you click the New button, Word assumes you want to start a new document based on the Normal template; it doesn't give you the chance to choose a different template).
Step 2: Choose a Tab Category

The New dialog box has several tab categories to choose from, such as Letters & Faxes. Depending on the type of document you want to create, click each tab and see what is available. (Depending on how Word was installed and whether anyone has created new templates, the tabs and templates you see may differ from those shown here).
Step 3: Preview a Template

When you locate a template you want to use, select it (by clicking just once on the icon); the Preview area lets you see what the design looks like.
Step 4: Open the Template

If you decide you like the template, double-click its name or select it and choose OK.
Step 5: Fill It In

Word creates a new document based on the template you chose. Many templates, such as the one shown here, include placeholder text with instructions to "Click here and type" to help you fill in your text. You may also see some crosshatched boxes. Word uses these for formatting purposes only, they won't print. Click the placeholder text and type your own text.
Step 6: Save, Print, and Close

The text you typed replaces the "click here" text. Continue replacing all the "click here" instructions with the text you want in the document. When you have completed the document, use the regular methods to save, print, and close it.

Insert Page Numbers
Step 1: Open Header & Footer

Click the View menu and then select Header and Footer. Word will switch to page layout view to display the header text area.
Step 2: Select Location

Click in the header or footer where you want to insert the page number. Use the tab key to move to the next tab stop.
Step 3: Format

Select the page number and format it as you would any other text.
Step 4: Close

When you’re done, click the Close button on the Header and Footer toolbar.

Track Changes
Step 1: Open Reviewing Toolbar

Click the View menu, select Toolbars and click Reviewing to display the toolbar.
Step 2: Save Version

Click the Save Version to save an unedited version of the document. Click the Track Changes button.
Step 3: Edit the Text

Edit the text. When you’re done, click the Previous Change button or the Next Change button to view changes.
Step 4: Accept or Reject

Click the Accept button or Reject button to manage the changes.

Create a Table
Step 1: Select Insertion Point

Move the insertion point to the beginning of a new paragraph.
Step 2: Click Insert Table

Click the Insert table button on the toolbar.
Step 3: Select Rows & Columns

Drag the mouse to select the correct number of rows and columns.
Step 4: Poof!

Release the mouse button to insert a blank grid on the document. The insertion point is the first cell, all ready for you to begin typing.

Format a Table
Step 1: Format Automatically

Select the table you want to format, click the Table menu and then select Table AutoFormat. Select a format from the Formats list and preview the results. When you find a format you like, click Apply.
Step 2: Align Text Horizontally

Select the cells, rows or columns you want to align and click the Align Left, Center, Align Right or Justify button on the formatting toolbar.
Step 3: Align Text Vertically

Click the Tables and Borders button on the toolbar to display the Tables and Borders Toolbar. Select the cells you want to vertically align, click an alignment button on the Tables and Borders toolbar.
Step 4: Change Text Direction

Click the Tables and Borders button on the toolbar to display the Tables and Borders Toolbar. Select the cells you want to change direction. Click the Change Text Direction button until the text is in the direction you want.

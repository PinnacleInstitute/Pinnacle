Technical Requirements

Netscape or Internet Explorer (versions 4.0 or higher).   Browsers should be capable of accepting cookies; SVGA Hi-color mode 800x600 resolution display.  Macromedia Shockwave 6.02 or higher.

Course Requirements

Before taking this course, the user should be familiar with the basic capabilities of Excel 2002 and should be able to use various tools and features provided in Excel for storing, viewing, and manipulating data.

Course Goals

Import pictures in Excel Insert and manage Clip Art Insert and format different types of drawing objects Create diagrams and organization charts Represent worksheet data in the form of charts

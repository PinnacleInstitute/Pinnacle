Technical Requirements

IBM-compatible PC with 166 MHz or higher processor, 32 MB RAM (64 MB RAM recommended); 800x600 SVGA color monitor with 16-bit high color display; Internet Explorer 4 or Netscape Navigator 4 or later with Shockwave 7.0.1; Windows 95, Windows 98, Windows NT; 3.5 diskette (for tracking); 28.8 kbps connection or higher for on-line use.

Course Requirements

Individuals taking this course should have a basic understanding of the MS Windows environments.

Course Goals

Create, edit, preview, print, save, and close a Word document
Select, copy, move, find, and replace text in a document
Correct spelling and grammatical errors in a document
Change font and font sizes, apply special font effects, borders and shading, and change paragraph alignment and spacing
Create bulleted and numbered lists
Create, modify, and format tables, and calculate and sort data in a table
Insert and modify headers and footers
Change document margins and page orientation, insert page breaks and page numbers, and create sections and columns

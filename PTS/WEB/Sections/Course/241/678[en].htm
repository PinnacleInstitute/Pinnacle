Changing Cell Alignment

When you enter information into cells, text is automatically aligned with the left edge of the cell and numbers are automatically aligned with the right edge of the cell. You can change the alignment of text and numbers to improve the appearance of your worksheet.

Text and numbers can be left-aligned, right-aligned, or centered in a cell. You may want to right align text at the top of a column of numbers to match the alignment of the numbers. Alignment changes are attached to the cell and affect any data entered into it.

Certain number styles control the alignment of numbers. Therefore, you may not be able to change the way some formatted numbers are aligned. If you format a number using the Currency Style or Comma Style button, you will not be able to change the alignment of that number. These buttons apply the Accounting style, which includes an alignment that cannot be changed. You can, however, change the alignment of numbers formatted with a Number format, and all text, regardless of formatting.

Using the Merge and Center Button

You can select several cells, merge them into one cell, and center the entry in that cell over the columns in a worksheet. This option is performed using the Merge and Center button and is useful when you want to center a title above specified columns in a worksheet.

Using the Borders Button

Cell borders are lines you can place on the edges of the cells in a worksheet. For example, you may want the column headings, row titles, and row totals of a worksheet to appear with wider lines to emphasize these cells. Excel contains twelve border styles that place lines of varying widths at different edges of the cells. In addition, you can add cell borders to an entire range of cells. Borders are attached to the cells and appear regardless of the cell entry.

The Borders button has two parts. You can use the list to display a list of border styles and choose a desired border style. The display on the button changes to display the currently selected border. To apply the currently selected border to other cells, you can click the button.

Using the Fill Color Button

You can add shading to the background of cells using the Fill Color button. For example, you can use shading to make the column headings distinct from the rest of the worksheet. Although color displays on the screen, it will not print unless you have a color printer. Colors will print as shades of gray on a black and white printer.

The Fill Color button has two parts. You can use the list to display the color palette and choose the desired color. The display on the button changes to display the currently selected color. To apply the currently selected color to other cells, you can click the button.

Using the Format Painter Button

The Format Painter button copies formats from a cell or range to another cell or range. This button prevents you from having to individually apply the formats to each cell or range every time you want to format cells. For example, if you apply bolding, italics, underlining, and shading to a cell, you can apply all of these formats, at one time, using the Format Painter button.

Copying to Non-adjacent Cells

You can use the Format Painter button to copy cell formatting to several non-adjacent cells or ranges at once. You do not have to reselect the Format Painter button each time you want to format a non-adjacent cell or range. For example, you may want to copy all the formats from range A1:A5 to both A10:A15 and A20:A25.

Clearing Formats

You can clear all the formats in a cell or range all at once. Clearing formats removes all formatting attached to the cell or range, including number formats, font formats, cell borders, and shading. If you only want to clear a single format, such as bolding, you should remove that format individually.

Inserting Selected Cells

Blank cells can be inserted into selected areas of a worksheet. This feature is helpful if you keep different types of data in different areas of the same worksheet, and need to insert a row or column in one section of data without affecting the rest of the worksheet.

When a full row is inserted, all columns in the worksheet are affected. Conversely, when a full column is inserted, all rows in the worksheet are affected. Inserting selected cells only affects the selected rows or columns in the worksheet.

When you insert cells, you are given the choice of shifting the surrounding data to the right or down. Shifting the data to the right affects the columns in which the data will appear. Shifting the data down affects the rows in which the data will appear. Therefore, if you insert cells B7:D7 and shift the data down, a new row 7 will be inserted in columns B, C, and D only. Though you can change the shift selection, Excel usually selects the correct shift alternative based upon your selection.

Deleting Selected Cells

Selected cells can be deleted from a worksheet. This feature is valuable if you keep different types of data in different areas of the same sheet, and need to delete a row or column in one section of data without affecting the rest of the worksheet.

Unlike deleting full rows and columns, where the entire row or column is deleted, deleting selected cells only affects the selected rows or columns in the worksheet.

When you delete cells, you are given the choice of shifting the surrounding data to the left or up. Shifting the data to the left affects which columns the data will appear. Shifting the data up affects which rows the data will appear. If you delete cells C9:E9 and shift the data up, row 9 will be deleted from columns C, D, and E only. Though you can change the shift selection, Excel usually selects the correct shift alternative based upon your selection.

Previewing a Worksheet

Before printing, you can preview a worksheet to see how the data appears on each page. The Print Preview feature displays the worksheet in the preview window. This window shows how the worksheet will look when it is printed, including all aspects of the layout. You can use the preview window to view the appearance of your worksheet before you print it. This window enables you to make any changes before printing your worksheet, if desired.

The default view in the preview window displays the full page, making the print difficult to read; however, you can increase the magnification of the worksheet by zooming the preview window. This option makes the data in the preview window easier to read. When the mouse pointer is positioned over the worksheet, it changes into a magnifying glass. When you click the worksheet with the magnifying glass, the magnification increases so that you can read the area of the worksheet you clicked. When you click the worksheet again, the magnification returns to full page view.

If your worksheet spans several pages, you can use the Next and Previous buttons in the preview window to view other pages.



Print Preview button

Read through the steps:

If you would like to complete these steps in an actual file, create a new file in Excel and then proceed through the steps.

Practice the Skill
  1. Click the Print Preview button on the Standard toolbar.
    The worksheet appears in the preview window with the page number displayed at the bottom left corner of the status bar.

  2. Click the area of the worksheet you want to magnify.
    The worksheet appears larger on the screen.

  3. Click anywhere in the worksheet to return to full page view.
    The entire worksheet appears on the screen.

  4. Select Next to view the next page for a multiple page worksheet.
    The next page appears in the preview window.

  5. Select Previous to view the previous page for a multiple page worksheet.
    The previous page appears in the preview window.

  6. Select Close .
    The preview window closes and worksheet view appears.

Printing the Current Worksheet

You can print a worksheet quickly using the Print button on the Standard toolbar. When you use this button, the current worksheet is sent directly to the printer, utilizing the current print settings. This button is useful when you need to print one copy of a worksheet quickly.

In addition to sending a worksheet to the printer, you can send a copy of a worksheet or workbook to another person via e-mail. Depending upon your mail application, you can send the worksheet or workbook as an attachment in the message or the worksheet or workbook as the message itself.

Printing the Selected Range

When printing a worksheet, you can choose to print just a selected range. This option is useful if you made changes to a specific range after the entire document has been printed. This option is available in the Print dialog box, along with options to print the active workbook and the entire workbook.


Read through the steps:

If you would like to complete these steps in an actual file, create a new file in Excel and then proceed through the steps.

Practice the Skill
  1. Select the range you want to print.
    The range is selected.

  2. Select the File menu.
    The File menu appears.

  3. Select the Print command.
    The Print dialog box opens.

  4. Select the Selection option under Print what .
    The Selection option is selected.

  5. Select OK .
    The Print dialog box closes and Excel prints the selected range.

Printing Multiple Copies

You may want to print multiple copies of a worksheet at one time. In Excel, you can select the number of copies you want to print in the Print dialog box.


Read through the steps:

If you would like to complete these steps in an actual file, create a new file in Excel and then proceed through the steps.

Practice the Skill
  1. Select the File menu.
    The File menu appears.

  2. Select the Print command.
    The Print dialog box opens.

  3. Enter the number of copies you want to print in the Number of copies spin box.
    The number appears in the Number of copies text box.

  4. Select OK .
    The Print dialog box closes and Excel prints the specified number of copies of the worksheet.

Printing a Page Range

You can choose to print certain pages of a worksheet in the Print dialog box. This option is useful when you make corrections to specific pages and you only want to print these corrected pages, rather than printing the entire worksheet.


Read through the steps:

If you would like to complete these steps in an actual file, create a new file in Excel and then proceed through the steps.

Practice the Skill
  1. Select the File menu.
    The File menu appears.

  2. Select the Print command.
    The Print dialog box opens.

  3. Select the From text box.
    The insertion point appears in the From text box.

  4. Type the page number from which you want to start printing.
    The page number appears in the From text box.

  5. Select the To text box.
    The insertion point appears in the To text box.

  6. Type the page number at which you want to stop printing.
    The page number appears in the To text box.

  7. Select OK .
    The Print dialog box closes and Excel prints the specified pages of the worksheet.

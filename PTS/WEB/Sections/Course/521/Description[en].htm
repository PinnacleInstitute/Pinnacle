Technical Requirements

Shockwave 8.0, 200 MH Pentium CPU (266 recommended), 32 MB of available RAM, (64 recommended), Windows 95, 98, 2000 or NT; (Windows 2000 or NT require 64MB of RAM)

Course Requirements

Previous experience using a PC is not required.

Course Goals

Feel comfortable and confident using a computer.  Learn the language of computers (e.g., proper terminology for parts).  Develop existing knowledge and basic skills.  Have a solid foundation for future learning.

Technical Requirements

Netscape or Internet Explorer (versions 4.0 or higher).  Browsers should be capable of accepting cookies; SVGA Hi-color mode 800x600 resolution display.  Macromedia Shockwave 6.02 or higher.

Course Requirements

Before taking this course the user should be familiar with the first course of FrontPage 2000.

Course Goals

Create and edit imagemaps for your Web Add animation to your Web page  Identify the use of audio with your FrontPage Web  Identify the use of video with your FrontPage Web Add FrontPage component on your Web page

Using the Crosstab Query Wizard

You can use crosstab queries to group and summarize information and display it in a spreadsheet format. Crosstab queries create less repetition of information in the datasheet, making it easier to read and analyze the selected field data. The Crosstab Query Wizard can help you quickly create a crosstab query. Through a step by step process, the Crosstab Query Wizard asks you for information about the tables and fields you want in your query.

You need three fields to create a crosstab query. The first field displays its values as row headings for the datasheet, the second field displays its values as column headings for the datasheet. The third field is the field on which the calculation is performed.

For example, crosstab queries can provide information as to how many of each product (listed as row headings) are sold by each sales representative (listed as column headings). At the intersection of each row and column is the amount of each product sold by each sales representative.

Read through the steps: . If you would like to complete these steps in an actual file, create a new file in Access and then proceed through the steps.

Practice the Skill
  1. Select the New button on the Database window toolbar.
    The New Query dialog box opens.

  2. Select Crosstab Query Wizard .
    Crosstab Query Wizard is selected and a description of the Crosstab Query Wizard appears in the left panel of the New Query dialog box.

  3. Select OK .
    The New Query dialog box closes and the Crosstab Query Wizard opens.

  4. Select the desired table or queryfrom the Which table or query contains the fields you want for the crosstab query results? list box.
    The table or query is selected.

  5. Select Next .
    The next page of the Crosstab Query Wizard appears.

  6. Add the desired field(s) you want to use to provide values for the row headings from the Available Fields list box.
    The field(s) appear in the Selected Fields list box and a sample of the row headings appears in the Sample box.

  7. Select Next .
    The next page of the Crosstab Query Wizard appears.

  8. Add the desired field(s) you want to use to provide values for the column headings from the Which field's values do you want as column headings? list box.
    The field(s) are selected.

  9. Select Next .
    The next page of the Crosstab Query Wizard appears.

  10. Select the field you want to use to calculate at the intersection point from the Fields list box.
    The field is selected, the corresponding functions appear in the Functions list box, and a sample of the field you want to calculate appears in the Sample box.

  11. Select the desired function from the Functions list box.
    A sample of the function appears in the Sample box.

  12. Select Next .
    The next page of the Crosstab Query Wizard appears with the insertion point in the What do you want to name your query? text box.

  13. Type a name for the query.
    The name appears in the What do you want to name your query? text box.

  14. Select Finish .
    The Crosstab Query Wizard closes, the query runs, and the RecordSet appears in Datasheet view.

Using the Find Duplicates Query Wizard

The Find Duplicates Query Wizard assists you in designing a query that displays duplicate records in tables. This query can be used to group records with like field values, such as finding all customers in a particular city. More importantly, there may be identical records stored in a table when only one record is necessary. The Find Duplicates Query Wizard displays these duplicate records. The unnecessary duplicate records can then be deleted from the table.

Read through the steps: . If you would like to complete these steps in an actual file, create a new file in Access and then proceed through the steps.

Practice the Skill
  1. Select the New button on the Database window toolbar.
    The New Query dialog box opens.

  2. Select Find Duplicates Query Wizard .
    Find Duplicates Query Wizard is selected and a description of the Find Duplicates Query Wizard appears in the left panel of the New Query dialog box.

  3. Select OK .
    The New Query dialog box closes and the Find Duplicates Query Wizard opens.

  4. Select the table or query on which you want to base the query from the list box.
    The table or query is selected.

  5. Select Next .
    The next page of the Find Duplicates Query Wizard appears.
  6. Add the desired field(s) in which you want to find duplicates from the Available fields list box.
    The field(s) appear in the Duplicate-value fields list box.

  7. Select Next .
    The next page of the Find Duplicates Query Wizard appears.

  8. Add any additional fields you want to show in the query from the Available fields list box.
    The field(s) appear in the Additional query fields list box.

  9. Select Next .
    The next page of the Find Duplicates Query Wizard appears with the insertion point in the What do you want to name your query? text box.

  10. Type a name for the query.
    The name appears in the What do you want to name your query? text box .

  11. Select Finish .
    The Find Duplicates Query Wizard closes, the query runs, and the RecordSet appears in Datasheet view .

Using the Find Unmatched Query Wizard

You can manage related data in tables using the Find Unmatched Query Wizard. This wizard assists you in designing a query that finds records that are "orphans," meaning that there are no records in a related table to which they correspond. For example, suppose you delete records from a Customers table containing the names of customers who have not placed any orders for more than one year. Two years later, an old customer places an order, which is entered into the Orders table. Because these tables are likely to be related, having an order without any customer information (such as the mailing address) is likely to cause problems. You can use the Find Unmatched Query Wizard to display these records and correct the problem by updating the Customers table.

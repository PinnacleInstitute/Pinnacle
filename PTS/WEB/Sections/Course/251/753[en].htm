Changing Row Height and Column Width

The standard height for rows in a table is 12.75. You can change the row height to allow field entries that are too long to wrap. This option allows you to display more of the data. Changing the height of a single row changes the height of every row in the table.

The standard width for columns in a table is 15.6667. You can change the column width to display any data in a field that is truncated. By widening the column, you allow more of the data to appear in the display.

Practice the Skill
  1. Point to a line between any two record selectors.
    The mouse pointer changes into a black cross with vertical arrowheads.

  2. Drag the line to the desired height.
    A line indicating the new row height moves as you drag and the height of all the rows changes.

  3. Point to the separator line to the right of the column header for the column width you want to adjust.
    The mouse pointer changes into a black cross with horizontal arrowheads.

  4. Drag the line to the desired width.
    A line indicating the new column width moves as you drag and the width of the column changes.

Changing a Font Attribute

You can change font attributes in Datasheet view using the Font Command dialog box. Changing font attributes changes the display of the data and the column headings. In the Font Command dialog box, you can change the font type, style, and size, as well as any font effects.

Practice the Skill
  1. Select the Format menu.
    The Format menu appears.

  2. Select the Font command.
    The Font dialog box opens.

  3. Change font attributes as desired.
    A preview of the changes appears in the Sample box.

  4. Select OK.
    The Font dialog box closes and the font attributes change.

Changing a Cell Effect

You can change the way cells appear in Datasheet view. For example, you can make cells raised or sunken. You can also change the look of the horizontal and vertical gridlines, as well as the background of cells. The Sample box in the Datasheet Formatting dialog box displays the results of your changes.

 

Practice the Skill
  1. Select the Format menu.
    The Format menu appears.

  2. Select the Datasheet command.
    The Datasheet Formatting dialog box opens.

  3. Change the data sheet formatting as desired.
    A preview of the changes appears in the Sample box.

  4. Select OK.
    The Datasheet Formatting dialog box closes and the cell effect changes.

Selecting a Column

Before you can manipulate entire columns in a table, you must select them. You can perform a number of actions on columns once they are selected. You can select a single column or multiple adjacent columns.

 

Practice the Skill
  1. Point to the column header of the column you want to select.
    The mouse pointer changes into a black arrow pointing downward.

  2. Select the column header.
    The column is selected.

Moving a Column

You can move columns in Datasheet view. For example, you can reposition columns to view data more easily or organize it more logically. You can move a single column or multiple adjacent columns. You must select the column or columns before you can move them.

 

Practice the Skill
  1. Point to the column header(s) of the column(s) you want to move.
    The mouse pointer changes into a black arrow pointing downward.

  2. Select the column header(s).
    The column(s) are selected.

  3. Drag the column(s) to the new location.
    A line indicating the new column(s) location moves as you drag and the column(s) appear in the new location.

Hiding/Unhiding a Column

If you do not intend to use certain fields, you can choose not to display them on the desktop by hiding the appropriate column(s). You can hide a single column or multiple adjacent columns.

Practice the Skill
  1. 1. Point to the column header(s) of the column(s) you want to hide.
    The mouse pointer changes into a black arrow pointing downward.

  2. Select the column header(s).
    The column(s) are selected.

  3. Select the Format menu.
    The Format menu appears.

  4. Select the Hide Columns command.
    The column(s) are hidden.

You can display hidden columns. Hidden columns are listed in the Unhide Columns dialog box. If a deselected check box appears beside a field name, it means that the column is hidden. You can select the check box to unhide the column.

Practice the Skill
  1. Select the Format menu.
    The Format menu appears.

  2. Select the Unhide Columns command.
    The Unhide Columns dialog box opens.

  3. Select the column(s) you want to unhide.
    The check box(es) of the column(s) are selected.

  4. Select Close.
    The Unhide Columns dialog box closes and the hidden column(s) appear.

Freezing a Column

As you scroll to the right in Datasheet view, the far left columns, which usually contain identifying information about the records, scroll off the screen. You can freeze these columns in Datasheet view. This option allows you to see the information in the frozen columns no matter how many columns you scroll to the right.

 

Practice the Skill
  1. Point to the column header(s) of the column(s) you want to freeze.
    The mouse pointer changes into a black arrow pointing downward.

  2. Select the column header(s).
    The column(s) are selected.

  3. Select the Format menu.
    The Format menu appears.

  4. Select the Freeze Columns command.
    A solid line appears to the right of the selected column(s).

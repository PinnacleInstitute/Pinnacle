Inserting a Column

You may find it necessary to insert one or more columns in a table. When you insert a column, you add a new field to the table. You must specify the location for the new field within the table. Access sets a default field width, provides a default field name, and assigns a default data type ( Text ). You can change any of these default settings in Design view.

 

 

Practice the Skill
  1. Open the University Employees table in Datasheet view.

  2. Position the insertion point in the column to the left of which you want to insert the new column.
    Select the Address column.

  3. Select the Insert menu.
    The Insert menu appears.

  4. Select the Column command.
    A blank column is inserted in the table to the left of the Address column.

Changing a Column Name

A column name should clearly indicate the information that a column contains in order to make working with data easier. When you change the name of a column, you change the field name. When you are selecting a name for a column, keep in mind that you cannot use duplicate field names. In addition, column names can be up to 64 characters long, but shorter names are recommended.

 

 

Practice the Skill
  1. Open the Workshop Registration table in Datasheet view.

  2. Point to the column heading you want to change.
    Point to the Cost column header and wait for the mouse pointer to change into a black arrow pointing downward.

  3. Double-click the column heading.
    The Cost column heading is selected.

  4. Type the new column name.
    Replace the word Cost with the word Tuition.

  5. Press [Enter].
    The column name is changed.

Deleting a Column

There may be times when you want to delete a column in a table. When a column is deleted, the columns to the right automatically adjust to the left. If other database objects (such as lookup tables) contain references to a deleted field, they generate error messages unless they are deleted as well.

 

Practice the Skill
  1. Click in a field in the column you want to delete.
    The insertion point appears in the column.

  2. Select the Edit menu.
    The Edit menu appears.

  3. Select the Delete Column command.
    A Microsoft Access warning box opens, asking if you want to permanently delete the selected field(s) and all the data in the field(s).

  4. Select Yes to confirm the deletion.
    The Microsoft Access warning box closes and the column is deleted from the table.

Inserting a Lookup Column

A lookup column saves time when you are entering data in a table by reducing the amount of information you must type in a field. A lookup column eliminates the need to search for information each time you want to add a record by looking up data in another data source. It also increases data entry accuracy by providing a list of standard values from which you can choose.

The Lookup Wizard can assist you in creating a lookup column to suit your needs. The values in a lookup column can come from another table, a query, or your own list. For example, in one table, you may have a list of customers. In another table, you may have a list of customer types and information related to each customer type. In the customer table, you can use the data from the customer type table for a lookup column. When you are working in the customer table, you can choose the customer type from a list to make the entries more consistent.

During data entry, a drop-down arrow automatically appears in a lookup column. You can use the arrow to access a list of values, or you can type a value that is not on the list directly into the column.

 

Practice the Skill
  1. Open the Workshop Registration table in Datasheet view.

  2. Position the insertion point in the column to the left of which you want to insert a lookup column.
    Select the First Name Column.

  3. Select the Insert menu. The Insert menu appears.

  4. Select the Lookup Column command.
    The Lookup Wizard opens.

  5. Select the option indicating how you want the lookup column to get its values.
    Select I want the lookup column to lookup the values in a table or query

  6. Select Next.
    The next page of the Lookup Wizard appears.

  7. Select the table or query you want to use to provide the values for the lookup column.
    Select the Workshop List table.

  8. Select Next.
    The next page of the Lookup Wizard appears.

  9. Add the field(s) you want to include in the lookup column from the Available Fields list box.
    Click on the single Left arrow to add the WorkshopID field.

  10. Select Next.
    The next page of the Lookup Wizard appears.

  11. Point to the separator line to the right of the column header for the column width you want to adjust.
    The mouse pointer changes into a black cross with horizontal arrow heads.

  12. Double-click the separator line to the right of the column header to automatically adjust the column width.
    The column width adjusts.

  13. Select Next.
    The next page of the Lookup Wizard appears with the insertion point in the What label would you like for your lookup column? text box.

  14. Type a label for the lookup column.
    Type WorkshopID.

  15. Select Finish.
    The Lookup Wizard closes and the lookup column is inserted in the table in Datasheet view.

Inserting a Hyperlink Column

When you insert a column in a table, Access assigns the Text data type as the default. However, you can insert a hyperlink column in a table, rather than a text column. A hyperlink field contains an address or path to another location, such as a network server or a page on the Internet. After you insert a hyperlink column, you can type a hyperlink address directly into the hyperlink field. Each record can contain a different hyperlink address. When you type the hyperlink address, the text appears underlined and in a different color, indicating the hyperlink. When you insert a hyperlink column, the column name defaults to Field1 . You can change a hyperlink column name the same as you change a text column name.

 

 

Practice the Skill
  1. Position the insertion point in the column to the left of which you want to insert the new column.
    The insertion point appears in the column.

  2. Select the Insert menu.
    The Insert menu appears.

  3. Select the Hyperlink Column command.
    A blank column is inserted in the table to the left of the selected column.

  4. Change the new column name to Web Address.
    Double click the Column heading and type "Web Address."

  5. Notice that entries will appear underlined and in a different color, indicating the hyperlink.

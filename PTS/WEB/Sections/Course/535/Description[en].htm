Technical Requirements

Netscape or Internet Explorer (versions 4.0 or higher).  Browsers should be capable of accepting cookies;  SVGA HI-color mode 800x600 resolution display.  Macromedia Shockwave 6.02 or higher.

Course Requirements

Before taking this course, the student should be familiar with the fundamentals of project management and planning work.

Course Goals

define project management list various project management related tasks list the various components of the Project 2000 window open an existing file specify project information for a new project

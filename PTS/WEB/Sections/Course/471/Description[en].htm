Technical Requirements

IBM-compatible PC with 166 MHz or higher processor, 32 MB RAM (64 MB RAM recommended); 800x600 SVGA color monitor with 16-bit high color display; Internet Explorer 4 or Netscape Navigator 4 or later with Shockwave 7.0.1; Windows 95, Windows 98, Windows NT; 3.5 diskette (for tracking); 28.8 kbps connection or higher for on-line use.

Course Requirements

Microsoft Excel 2000 Basic.

Course Goals

Manage workbooks and prepare them for the Web
Record, run and edit macros
Create,sort, print, and summarize data from lists
Find and delete records
Use AutoFilter and create custom and advanced filters
Customize charts and map data
Set up and use a shared workbook
Create interactive worksheets and PivotTables for the Web
Create hyperlinks between Excel files and the Web
Run queries to retrieve data from the Web

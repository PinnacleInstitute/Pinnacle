Technical Requirements

Ability to use Microsoft Word, or some other such word processing package, although it is not necessarily important if the user does not have access to word processing.

Course Requirements

Basic understanding of the English language.

Course Goals

To improve the quality, vocabulary and grammatical points of the English language that the individual uses.

Course Materials

Poems, literature, extracts from novels etc.

Grading Policy

Self-determined, although there are short question sections at the end of each lesson to summarise.

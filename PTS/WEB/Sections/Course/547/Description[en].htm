Technical Requirements

Netscape or Internet Explorer (versions 4.0 or higher).   Browsers should be capable of accepting cookies; SVGA Hi-color mode 800x600 resolution display.  Macromedia Shockwave 6.02 or higher.

Course Requirements

The users should be familiar with the Excel interface, data formatting options, and use of formulas. Additionally, the users should have basic knowledge of databases.

Course Goals

Build, sort, and filter lists Validate data and calculate subtotals Use functions to analyze list information Import data from different data sources Work with external data queries

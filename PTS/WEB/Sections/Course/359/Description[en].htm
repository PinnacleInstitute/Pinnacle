Technical Requirements

Although you do not need a Group Exercise Certification to take this course, if you are going to be teaching kickboxing in a Group Fitness setting, then it is recommended that you have a Group Exercise Certification before you teach this format in a class setting. 

Course Requirements

You will be required to read all materials and complete all quizzes and written assignments for this course. 

Course Goals

To gain a basic understanding of a Group Exercise class and your role as an instructor/participant. 

To gain a basic understanding of kickboxing techniques and terminology. 

To be able to format a kickboxing class with or without the use of punching bags.

To be able to perform techniques properly and be able to cue them to group fitness members.

Set a Recurring Appointment
Step 1: Schedule a Recurring Appointment

To schedule a recurring appointment from the Calendar folder, first open the Appointment Recurrence dialog box. Display the Actions menu and select New Recurring Appointment.
Step 2: Enter a Time

Use the Start, End, and Duration drop-down lists to set the appointment start and end times and designate how long the appointment lasts.
Step 3: Enter a Recurrence Pattern

Under Recurrence pattern, select the frequency of the appointment and the day on which the appointment falls. Depending on your selection, the remaining recurrence options will vary. If you select Weekly for example, you can specify every week or every other week.
Step 4: Range of Recurrence

Use the Range of recurrence options to enter any limits to the recurring appointment. You may need to schedule five dentist visits over the next five months for example; after that, you no longer need the appointment. Use the End after option to set such a range.
Step 5: Fill in the Details

Click OK to close the Appointment Recurrence window and display the Appointment form. Now you can fill in any details about the appointment. Click the Save and Close button when you're finished.

Outlook Mail Services
Step 1: Welcome to the Inbox

The Inbox view is where you manage all your incoming, outgoing, and stored messages. When you click the Inbox icon in the Outlook bar, Outlook switches to this view and displays the contents of the Inbox folder. This is where new email goes when it arrives.
Step 2: Using the Folder List

Activate the Folder List (a list of all the folders in which you can store messages) by choosing View and selecting Folder List. To switch to a folder, just click that folder. You can move a message to a folder by dragging it from the folder contents pane on top of the folder name.
Step 3: Toolbar Message Actions

The toolbar is where you'll find the most common commands for dealing with email. In particular, the Reply and Reply to All buttons let you respond to an email sender. The Forward button lets you send a message that you've received to someone else.
Step 4: Toolbar Message Management

You'll also use a few other buttons on the toolbar frequently. Use the Send/Receive button to check for mail; use the Find button to search for a specific message by sender, date, keyword, or subject. The Organize button displays tools that make it easy to mark and move messages for easy reference.
Step 5: Understand the Message Symbols

You can quickly assess the status of messages in the Inbox view by looking at the symbols in the folder contents window. The Message Importance icon indicates a sender-assigned status. The arrow's direction in the Message icon tells you whether a message has been replied to or forwarded. You can mark messages with the Message Flag icon as a reminder that some action is required. The Attachment icon indicates that a message has files attached to it.
Step 6: Switch Views

The Outlook bar is always available to allow you to change the view to another part of Outlook. Just click the icon in the Outlook bar that represents the view you want to switch to.

Outlook Mail Profiles
Step 1: Overview

A collection of services and their settings are stored in Profiles. Profiles allow you to create multiple configurations of Outlook and to configure Outlook so that more than one user can access it.
Step 2: Access Profiles

Click the Start menu, select Control Panel and then double click on the Mail icon.
Step 3: Select a Profile

You can choose an existing profile from the Profile Name combo box at startup.
Step 4: Create a Profile

You can create a new profile by clicking on the New button and launching the Inbox Setup Wizard.

Send Mail with Outlook
Step 1: Use the Shortcut

You can send an e-mail message by pressing Ctrl+Shift+M from anywhere in Outlook or Ctrl+N while in the Inbox folder.
Step 2: Use the Menu

You can also select File, New, and Mail Message from the menu bar. Or click on the New button on the Standard toolbar.
Step 3: Destinations

First you need to indicate where the message is going. You can either click on the names in the To text box directly or click the To button and open the Names dialog box from where you can choose who to send the message to.
Step 4: Subject

All e-mail messages have a subject, which should tell the recipient what the message is about. Enter a subject in the Subject field, and then enter your message in the body of the email message.

Receive Mail with Outlook
Step 1: Inbox Views

When receiving and reading your mail, you will depend on the views of the inbox. Default views for the inbox include a basic messages view, which simply lists your messages in a table view allowing you to sort by any of the columns.
Step 2: AutoPreview

If you select a view that uses AutoPreview, you can see the first part of the message before opening it to help you determine if you actually want to read the message, or just delete it.
Step 3: Double-Click

When you want to read an e-mail message, double-click on it to open the message and read its full contents.
Step 4: Answer It

You can click the Reply button to send a reply to the message.

Create Hyperlinks in Documents
Step 1: Intro to Hyperlinks

Sometimes you’ll want to refer to another part of the same document or a file created in a different program. Rather than duplicating the information or adding a footnote, you can insert a hyperlink. A hyperlink is an object that you click to move to another location. The move can be within the same document, to a file on your computer, or to a website.
Step 2: Choose Location

Select the location from which you want the hyperlink to jump. Click where you want to insert the hyperlink and then select it.
Step 3: Insert the Hyperlink

Click the Insert Hyperlink button on the toolbar. Click the Named Location in File Select button, and then double click the name you want to jump to. Click OK when you’re finished.
Step 4: Hyperlink Between Documents

Position the insertion point where you want the hyperlink to appear. Type and then select the text you want to use as the hyperlink. Now, click the Insert Hyperlink button on the toolbar. Click the Link to File or URL Select button and then double click the file you want to jump to. Click OK when you are finished.

Use and Remove Hyperlinks
Step 1: Position the Mouse

Position the mouse pointer over any hyperlink. A popup window will appear telling you where the link goes.
Step 2: Click the Hyperlink

Click the hyperlink the screen will either jump to a new location within the same document, jump to a web site, or open a new file and the program that created it.
Step 3: Select Hyperlink

To remove a hyperlink, select the hyperlink you want to remove by dragging the pointer across the hyperlink to select it, but don’t click it.
Step 4: Remove Link

Click the Insert Hyperlink button on the Standard toolbar. Click the Remove Link button. Click OK when you’re finished.

Use the Web Toolbar
Step 1: Show or Hide

To show or hide the Web Toolbar click the Web Toolbars button on the main toolbar. This button toggles on and off to show and hide the Web Toolbar. Or you can select View, Toolbars, and Web Tools.
Step 2: Show or Hide Other Toolbars

Click the Show Only Web Toolbar button on the Web Toolbar. This will hide all of the other toolbars so you can only see the Web Toolbar.
Step 3: Start Page

With the Start Page button, you can jump to your start page with just one click.
Step 4: Change Your Start Page

Open the document you want as your start page. Click the Go button on the Web Toolbar. Click Set Start page, then click Yes to confirm the new start page.

Hyperlinked Documents
Step 1: Use the Address Box

You can jump directly to a document on your computer or network. Or to a web page using the Web Toolbar. In the Address box on the Web Toolbar enter the address for the document you want to view.
Step 2: Jump to a Document 1

Click in the Address box on the Web toolbar and select the current address. Type an address for the document and press return. You’ll are now taken to that document.
Step 3: Jump to a Web Document 2

Click in the Address box on the Web toolbar and select the current address. Type an Internet address, for example http://www.microsoft.com.
Step 4: Connect to the Internet

Connect to the Internet through your ISP. Your web browser will open and you’ll be taken to that site. When you’re finished, click the File menu and select Close.

Navigate a Hyperlinked Document
Step 1: Back

To back up one document, click the Back button on the Web toolbar.
Step 2: Forward

Click the Forward button on the Web toolbar until you return to the most recent document you opened.
Step 3: Jump to Any Hyperlinked Doc.

Click the Address drop down arrow. Click the document or address you want to jump to.
Step 4: Stop

Click Stop Current Jump button on the Web toolbar. To reload a document, click the Refresh Current Page button on the web toolbar.

Get Info From the Web
Step 1: Search

Click the Search the Web button on the Web toolbar. Connect to the Internet through your ISP. Your web browser will open.
Step 2: Select Provider

Click the Search tab. Click the Select Provider pop-up menu and then select a search engine.
Step 3: Select a Topic

Type in a topic you want to search for on the Internet. When you’re finished click the File menu and select Close.
Step 4: Change Your Search Page

Open the document you want as your search page. Click the Go button on the Web toolbar. Click Set Search Page. Click Yes to confirm the new search page.

Return to Favorite Documents
Step 1: Open File

Open the file, or Internet site that you want to access from the Favorites folder.
Step 2: Click Favorites

Click the Favorites button on the Web toolbar. Then click Add to Favorites.
Step 3: Jump to a Favorite

Click the Favorites button on the Web toolbar. Click the favorite document you want to jump to.
Step 4: Delete a Favorite

Click the Favorites button, click Open Favorites, select the favorite icon you want to delete and press the Delete key.

Create Web Documents
Step 1: Create a Document for the Web

Click the File menu and select New. Click the Web Page tab, and then double click the Web Page Wizard.
Step 2: Select Type

Select the type of web page you want to create and then click Next. Select the visual style you want to create and then click Finish.
Step 3: Create a Web Presentation

Click the File menu and select New. Click the General tab and then double click the Web Page Wizard.
Step 4: Select Type

Click Next, select the type of presentation you want to give and then click Next. Click the Internet Kiosk option button and then click Next. Enter an email address and then click Finish.

Save Web Documents
Step 1: Save a Word Document in HTML

Open the document you want to save as HTML. Click the File menu and then click Save As Web Page. Select the folder where you want to store the file. Enter a name for the file and then click Save.
Step 2: Save an Excel Document in HTML

Open the document you want to save as HTML. Click the File menu and then click Save As Web Page. Click Add to specify the ranges and charts to include on the Web page and then click Next.
Step 3: Excel Options

Click an option button to create a new Web page or add your data to an existing Web page and then click Next. Enter header and footer information and then click Next. Select the pathname of the file you want to save. Then click Save.
Step 4: Save a Presentation in HTML

Open the document you want to save as HTML. Click the File menu and then click Save As Web Page then click Next. Respond to the prompts that appear about page style and graphic type, and then click Finish.

Preview Web Documents
Step 1: Open a Browser

Double click the Internet icon on the desktop or open your default browser manually.
Step 2: Enter Address

Click in the Address box on the Address bar. Enter an address to the web page you want to preview. Press the Return key.
Step 3: Open Word

You can also preview a web page in Word. Double click the Microsoft Word icon in the Microsoft Office XP folder. Click the File menu and select Open Web Page.
Step 4: Enter Address

Type or select an address to the web page you want to preview and then click OK.

Technical Requirements

Netscape or Internet Explorer (versions 4.0 or higher).   Browsers should be capable of accepting cookies; SVGA Hi-color mode 800x600 resolution display.  Macromedia Shockwave 6.02 or higher.

Course Requirements

The users should be familiar with basic concepts and usage of Internet.

Course Goals

Describe the basic concepts related to searching on the Internet  Identify the guidelines for formulating a query Identify the Boolean operators which are used to combine keywords in a search query  Formulate a query

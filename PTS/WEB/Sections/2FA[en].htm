Install Application

We highly recommend that you add 2 Step verification to your account for additional
security protection. 2 Step Verification, also know as 2 Factor Authentication (2FA),
requires an additional security code to be entered along with your logon name and password.

You will need to install an application on your phone or computer that generates
a random security code every 20 to 60 seconds that is associated with
your account. After the application is installed you will need to setup the application
for your account by scanning the QR Code OR entering the Manual Setup Code below.


The following 2FA applications are available for your installation.


Authy (Preferred Multi-Platform Apps)

Mobile Apps: Apple (IOS) & Android
Desktop Apps: Apple, Windows and Linux


Google Authenticator (Alternative)

For Apple: Google Authenticator on iTunes App Store
For Android: Google Authenticator on Google Play
For Windows Phone: Authenticator on Windows Phone App Store
For Windows 8: Google Authenticator on Windows App Store
For Windows PC: WinAuth version 3.1 for Windows

There are others, but that should cover most users pretty well.

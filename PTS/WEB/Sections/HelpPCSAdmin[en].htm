Project Collaboration System

Administrator Guide

Introduction
The Pinnacle Project Collaboration System (PCS) is a project management tool designed to accomplish the following goals.

1.  Collaboration :  Enable a team of distributed individuals to work together on a shared project.
2.  Accountability : Track individual performance on assigned tasks.
3.  Simplicity : Provide an easy-to-use tool to centrally manage projects.

PCS provides total flexibility in defining the structure of any project from simple to complex. A project may consist of an unlimited number of sub-projects and each project or sub-project may have an unlimited number of tasks and sub-tasks. Collaboration is accomplished with an unlimited number of notes and attached documents at any level, and a discussion board for members to submit and reply to archived messages.

Administrator Setup
Before a company starts using the capabilities of PCS, it can setup a custom list of project types.  The company administrator can access PCS after signing into Pinnacle by selecting the “Projects” link under the “Manage Business” caption on the navigation bar on the left hand side of the page.

Project Types
Select the “Project Types” link in the upper right hand corner of the “Project Collaboration System” page. Existing project types are displayed in a list in sequence order. Click the name of the project type in the list to edit the existing project type. Some examples of project types may include; software, product, marketing, creative, etc.

Enable Member Access
Access to the Project Collaboration System must be enabled for each individual member.  The Pinnacle Account Executive can specify the default project access for newly added members.  The company administrator can specify each individual member’s project access on the member’s information page.  A member’s project access can be set to all projects that they are assigned to, or may be set to all projects with a security level equal to or less than their membership security level.  Security level access is convenient for management to view all projects without being added as a member of the projects.

Project Collaboration System

User Guide

Overview
The Pinnacle Project Collaboration System (PCS) is a project management tool designed to accomplish the following goals.

1.  Collaboration :  Enable a team of distributed individuals to work together on a shared project.
2.  Accountability : Track individual performance on assigned tasks.
3.  Simplicity : Provide an easy-to-use tool to centrally manage projects.

PCS provides total flexibility in defining the structure of any project from simple to complex. A project may consist of an unlimited number of phases and each project or phase may have an unlimited number of tasks and sub-tasks. Collaboration is accomplished with an unlimited number of notes and attached documents at any level, and a discussion board for members to submit and reply to archived messages.

Project Tracking
The Project Collaboration System (PCS) provides members with powerful project tracking capabilities and management reports. Each signed-in member that has the Project Collaboration System enabled can access it by clicking the “My Projects” link on the navigation bar on the left hand side of the page.

The PCS page provides links to all PCS functionality and the ability to search for projects by project name, description, project type or project #.  The project name and description can be searched for text anywhere within the field.  The searched list of projects can also be filtered by the project status option of all statuses, started, completed or cancelled.  

Click the project name to view the project detail page.

Tasks  (A Unit of Work)
Tasks can be added by clicking the “New Task” link at the top of the page.  A project, phase or task may contain an unlimited number of tasks that are displayed on the project or task page. Each task is listed with the name of the task, owner, status, start date and end date. Click the name of the task to view the task details. The task page lists all of its sub-tasks, documents and notes. Click the “Edit” button to edit the details of the task including the task’s owner, task name, sequence, description, projected start and end dates, actual start and end dates, projected cost, actual cost, and hours.

Documents
Documents can be added by clicking the “New Document” link at the top of the page.  A project, phase or task may contain an unlimited number of attached documents that are displayed on the project or task page. Each document is listed with the name of the document, the file link or URL link, status, file size, and date.  Only the user that attached the document may edit the document.  Any user can click the file link or URL link to view a popup window with the selected document content.

Notes
Unlimited notes can be added to any project, phase, task or sub-task.  The list of notes is displayed on the page with the date and time, user name that entered the note, and the content of the note.  New notes can be added by clicking the “Add” button above the note list. The notes are listed in date order with the most current notes at the top of the list.  The user that entered the note can edit the note by clicking the “edit” link following the note content.  Any note may be pinned to the top of the list by selecting the “Pin” option at the bottom of the note edit page.

Project Members
Each project and phase has its own list of included members.  Project members can be viewed and added by clicking the “Members” link at the top of the page. A list of project members is displayed with their name and status and a “remove” link to take them off the list.

New members may be included in the project by clicking the “Include Members” link in the upper right hand corner of the page.  A popup page enables the user to search for members to include on the project.  You can search by the member’s name, company name, member #, etc.  The search result list of members includes a checkbox in front of each member name.  Select the checkboxes of all members to be included in the project and click the “Include Checked Members” button to add them to the project.

Phases may include all members from the parent project’s member list by clicking the “Copy All Parent Project Members” button on the bottom of the page.

Discussion Board
If the project has the discussion board option selected, clicking the “Discussion Board” link at the top of the page will popup a page with a project specific discussion board.  New discussions may be added or replies may be made to existing discussions.  Click the “Add To Favorites” link on the discussion board to have the Training Buddy notify you if new messages have been posted since your last visit.  The Training Buddy is accessible from your home page or on the left-hand side navigation bar.

Conference Room
If the project has the conference room option selected, clicking the “Conference Room” link at the top of the page will popup a page with a project specific conference room.  The users can text chat with any other members that may be in the conference room at that time.

Project Type
The list of Project types is a custom list defined by each company.  Select the appropriate Project type or contact the company administrator for additional list options. The Project type is used for information purposes and for categorizing Projects for management reporting.

Milestone Dates
A project may optionally specify projected start and end dates.  If an entered projected date is modified later the system will automatically log a note documenting the change.   If a projected date and actual date is specified, the variance is automatically calculated and displayed if it is non-zero.  If the actual date is past the projected date, the variance is displayed as a positive red number.  If the actual date is before the projected date, the variance is displayed as a negative black number.

Project Costs
The project’s projected and actual costs for each task can be optionally specified.  The actual costs are automatically rolled up and summarized for the entire project.  Cost variances are displayed as red positive numbers that are over budget, or as black negative numbers that are under budget. Costs can be specified for any task or project, and are added to the accumulated total rolled-up costs of items below it.

Project Hours
The project’s hours for each task can be optionally specified.  The hours are automatically rolled-up and summarized for the entire project.  Total hours are displayed for the specific project or task.  Hours can be specified for any task or project, and are added to the accumulated total rolled-up hours of items below it.

Add New Project
The Project Collaboration Page that displays a list of searchable projects includes an “Add New Project” link at the top of the page.  The new project information can be entered OR an existing project may be copied by specifying the project # of the existing project at the bottom of the page.

The existing project to be copied is display with all of its members, tasks, and phases. A selected checkbox is displayed in front of each item to be copied.  Unselect the checkbox for each item that should not to be copied and click the “Copy” button.  The projected dates will be adjusted forward based on the current date.

Additional Member Options
Any of the above options can be added to the member options.

Excluding Member Options
Preface any of the above options with a negative sign to exclude the specified option from the member.

Menu Section Display Options 
Each of the text boxes following the options specifies which menu sections will be displayed open.  For example; "012" will show the first, second and third displayed menu sections opened.

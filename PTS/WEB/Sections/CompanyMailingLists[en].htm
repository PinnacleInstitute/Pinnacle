Company Email Lists

The following Company email lists have been created by Pinnacle for your use.  Email lists can be maintained with the Email Manager - "Mailing Lists" link in the upper right hand corner of the page.  The optional parameters must be provided as specified below. Each email list will return up to five additional data items identified as data1, data2, data3, data4 and data5 for use in the subject and body of the email message.

All Members Enrollment Date
List all members for the specified status and for the optioanlly specified enrollment date range.

Parameter #1:  Status  (0=all, 1=active, 2=trial, 3=certify, 4=suspended, 5=inactive)
Parameter #2:  From Date
Parameter #3:  To Date
Parameter #4:  n/a
Parameter #5:  n/a

Email:  Member Email
First Name:  Member First Name
Last Name:  Member Last Name
Data #1:  Member Status
Data #2:  Enrollment Date 
Data #3:  n/a 
Data #4:  n/a
Data #5:  n/a

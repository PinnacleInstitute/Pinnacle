Product Input Options

You can define custom fields for each individual product. Each user-defined Input field starts on a separate line and can continue to the next line if too long. The field name must be followed by a colon(:). If the name of the field starts with an Asterisk(*), entry is required. If the name of the field starts with a hyphen(-), it is only visible to employees and company administrators.  If the name of the field starts with an underscore (_), it can be edited by employees and company administrators, but displayed only to others.  If an asterisk and hyphen or underscore) are both specified, the asterisk must be first (e.g. *-Size).

A list of choices must be separated by semi-colons(;). If the name of the choice starts with an Asterisk(*) it will be the default choice. A choice can be followed by a price in parathesis (2.00).

Non-list fields can specify the data type (type=text (default) | memo | number | date), minimum and/or maximum values (min=xxx max=xxx), price in parathesis (3.00), width of text box or height of memo (size=xx), and a default value (value=abc). If a default value is specified it cannot include any spaces. If type=date, the min, max and value settings can be entered as NOW for the current date.

A prompt can be included with the field name for input instructions (Size[Enter your personal size]:) and cannot include a colon.

EXAMPLES

*Size: Small; Medium; Large; Extra Large ($2.00)  

*Color: *Red; Green; Blue  

Monogram[Enter up to 3 characters]: type=text max=3 size=3  ($3.00)
